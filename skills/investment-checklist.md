# 버핏 가치투자 매수 전 Checklist

$ARGUMENTS 에 대해 버핏 가치투자 매수 전 Checklist 분석을 실행한다.

**지원 입력 형식**: 단일 또는 복수 회사, 쉼표/모점/공백으로 구분. 예: `텐센트, 마오타이, 엔비디아` 또는 `NVDA AAPL MSFT`

## 실행 흐름

### 1단계: 입력 파싱, 분석 대상 회사 전부 식별

$ARGUMENTS 에서 모든 회사명/종목코드를 파싱한다. 각 회사에 대해 다음을 확정한다:
- 회사 정식 명칭, 종목코드, 상장 거래소
- 회사가 비상장이면 "비상장"으로 표기하고 간략한 설명(간접 투자 경로가 있는지)을 제시한 뒤, 전체 Checklist는 건너뛴다

### 1.5단계: AI 연구 편향 사전 경고

각 회사에 대해 "정보 풍부도" 신속 등급(A/B/C)을 매기고, 보고서에 표기한다:

| 등급 | 판단 기준 | Checklist에 미치는 영향 |
|------|---------|-----------------|
| A급 | 다년간 상장, 데이터 충분 | 정상 실행, 단 "컨센서스 함정" 경계 —— 모든 지표가 명확해 보인다고 정말 확실한 것은 아니다 |
| B급 | 데이터 제한적, 추산 필요 | 추산한 지표마다 신뢰도 표기, "좋은 사업" 판단 시 데이터 신뢰성을 가중 고려 |
| C급 | 정보 극도로 희소 | 6관문 표를 억지로 채우지 말고, "데이터 부족으로 판단 불가"를 정직하게 표기, 검증 가능한 핵심 문제에 집중 |

**핵심 원칙**: Checklist의 목표는 **나쁜 선택을 배제하는 것**이다. C급 회사의 경우 "데이터 부족"은 "불통과"도 아니고 "통과"도 아니다 —— "회색 지대, 1차 정보 보완 필요"로 정직하게 표기해야지, AI가 표를 채울 수 없다고 해서 부결로 판정해서는 안 된다.

돤융핑(段永平)은 이렇게 말했다: "이해할 수 없다"에는 두 종류가 있다 —— 하나는 사업이 너무 복잡해서 정말 이해할 수 없는 것이고, 다른 하나는 아직 시간을 들여 들여다보지 않은 것이다. AI 연구의 한계는 "자료가 적은 것"과 "이해할 수 없는 것"을 혼동하기 쉽다는 점이다.

### 2단계: 병렬 데이터 수집

Task 도구로 **각 회사**마다 독립된 백그라운드 Agent를 가동해 데이터를 수집한다(모든 회사를 동시에 병렬 가동). 각 Agent는 다음 수집을 담당한다:

1. **수익성**: ROE(5-10년 추세), 매출총이익률, 순이익률, 잉여현금흐름
2. **밸류에이션 데이터**: 현재 주가, 시가총액, PE(TTM), 선행 PE, PB, 배당수익률
3. **성장 추세**: 최근 3년 매출/이익 증가율
4. **재무 건전성**: 부채 수준, 자본적지출 필요, 현금 보유, 순현금/순부채
5. **경쟁 구도**: 시장 점유율, 주요 경쟁자, 점유율 변화 추세
6. **해자 증거**: 브랜드/전환비용/네트워크 효과/규모의 경제/기술 장벽의 구체적 증거
7. **경영진 기록**: CEO 경력, 핵심 결정, 지분, 자본 배분 기록
8. **최신 동향**: 최근 6개월 중대 사건(실적, M&A, 규제, 경영진 변동 등)

### 3단계: 회사별 6관문 Checklist 실행

각 상장 회사에 대해 순서대로 6개 관문을 통과시킨다:

---

#### 제1관: 나는 이 사업을 이해할 수 있는가(능력범위)

반드시 답해야 한다:
- [ ] 이 회사가 무엇으로 돈을 버는지 한 문장으로 명확히 말할 수 있는가?
- [ ] 10년 후에도 높은 확률로 여전히 무슨 사업을 하고 있을까?
- [ ] 어떤 핵심 변수가 성패를 결정하는가?
- [ ] 이 산업에 대한 인식이 심층 연구에서 온 것인가, 풍문에서 온 것인가?

**평가 기준**(★1-5):
- ★★★★★: 비즈니스 모델이 극히 단순명료, 10년 확실성 높음(예: 마오타이 —— 술 빚어 술 팖)
- ★★★★☆: 모델은 명확하나 기술 문턱이 있어 일정 전문 지식 필요
- ★★★☆☆: 모델은 이해 가능하나 10년 확실성 낮음, 산업 변화 빠름
- ★★☆☆☆: 사업 라인이 복잡하거나 산업이 격변 중, 미래 예측 어려움
- ★☆☆☆☆: 완전히 능력범위 밖

**강제 부결**: 돈 버는 방식조차 말할 수 없다면, 바로 "능력범위 밖, 분석 안 함"으로 표기한다.

---

#### 제2관: 이것은 좋은 사업인가(경제적 특성)

데이터로 말하라, **핵심 지표는 반드시 도구로 정밀 계산해야 한다**:

```bash
python3 ~/ai-berkshire/tools/financial_rigor.py verify-valuation \
  --price {주가} --eps {EPS} --bvps {주당순자산} --fcf-per-share {주당FCF} --dividend {주당배당}
```

| 지표 | 해당 회사 수치 | 참고 기준 | 판단 |
|------|-----------|---------|------|
| ROE(5년 평균) | | >15% 우수, >20% 탁월 | |
| 매출총이익률 | | >40%면 가격결정력 암시 | |
| 잉여현금흐름 | | 지속적 플러스, ≈순이익 | |
| 자본적지출 강도 | | 경자산이 중자산보다 우수 | |
| 부채 수준 | | 이자부 부채/순이익<3년 | |

**평가 기준**(★1-5):
- ★★★★★: ROE>25%, 고마진, 강한 FCF, 경자산, 저부채(전부 충족)
- ★★★★☆: 4개 항목 충족
- ★★★☆☆: 3개 항목 충족
- ★★☆☆☆: 2개 항목 충족 또는 추세 악화
- ★☆☆☆☆: 대부분 미충족 또는 FCF 지속 마이너스

---

#### 제3관: 해자가 충분히 깊은가(경쟁우위)

항목별로 점검한다:

| 해자 유형 | 보유 여부 | 구체적 증거 | 넓어지는가 좁아지는가? |
|-----------|---------|---------|--------------|
| 브랜드/가격결정력 | | | |
| 전환비용 | | | |
| 네트워크 효과 | | | |
| 비용/규모 우위 | | | |
| 기술/특허 장벽 | | | |

추가 검증: 경쟁자에게 100억을 준다면, 이 사업을 복제할 수 있는가?

**평가 기준**(★1-5):
- ★★★★★: 다중 해자가 중첩되고 넓어지는 중
- ★★★★☆: 최소 하나의 강한 해자가 안정적
- ★★★☆☆: 해자는 있으나 충분히 깊지 않거나 추세 불명
- ★★☆☆☆: 해자가 침식되는 중
- ★☆☆☆☆: 뚜렷한 해자 없음

---

#### 제4관: 경영진은 신뢰할 만한가(사람의 요소)

| 점검 항목 | 평가 |
|--------|------|
| 정직성(약속 vs 이행) | |
| 자본 배분 능력(자사주 매입/배당/M&A 기록) | |
| 주주 이익 지향(지분, 보수) | |
| 주인의식(창업자 vs 전문경영인) | |
| 기업 지배구조(특수관계인 거래, 영업권, 감사) | |
| CEO가 떠나도 정상 운영될 수 있는가? | |

**평가 기준**(★1-5):
- ★★★★★: 창업자가 키를 잡고, 자본 배분이 탁월, 이익이 완전히 일치
- ★★★★☆: 경영진이 우수하나 작은 흠 있음
- ★★★☆☆: 경영진은 합격이나 지배구조 우려 있음
- ★★☆☆☆: 정직성 또는 지배구조 문제 있음
- ★☆☆☆☆: 심각한 정직성 문제(→강제 부결)

| 지표 | 수치 | 역사적 분위 | 판단 |
|------|------|---------|------|
| PE (TTM) | | | |
| 선행 PE | | | |
| PB | | | |
| 배당수익률 | | | |
| FCF Yield | | | |

추가 검증(**반드시 도구로 정밀 계산, 암산 금지**):

- ★★☆☆☆: 비싼 편, 안전마진 부족
- ★☆☆☆☆: 심각한 고평가

#### 제6관: 포지션과 의사결정 규율(감정 폭주 방지)

다음 감정 신호를 점검한다:
- FOMO 때문에 사고 싶은가?
- 남이 추천했다는 이유로 사고 싶은가?
- 5년간 거래정지되어도 받아들일 수 있는가?
- 매수 논리를 200자 이내로 명확히 쓸 수 있는가?

### 4단계: 거울 테스트

각 회사에 대해 거울 테스트 문장을 써낸다:

> "나는 ___원에 ___회사를 매수한다, 그 이유는:
> 1. 이 사업의 본질은 ___이고, 나는 그것을 이해한다;
> 2. 이 회사의 해자는 ___이며, 넓어지는/좁아지는 중이다;
> 3. 경영진은 ___, 신뢰할 만하다/만하지 않다;
> 4. 현재 가격은 내재가치의 ___할에 해당하며, 충분한 안전마진이 있다/없다;
> 5. 설령 내가 틀렸어도, 하방 리스크는 통제 가능하다/불가능하다, 왜냐하면 ___."

**5문장으로 완성하지 못하면 = 사지 않는다.** "통과" 또는 "미통과"를 명확히 표기한다.

---

### 5단계: 신속 부결 체크리스트

각 회사에 대해 항목별로 점검하고, 어느 하나라도 발동되면 바로 "부결"로 표기한다:

- [ ] 이 회사가 어떻게 돈을 버는지 명확히 말하지 못한다
- [ ] 3년 연속 잉여현금흐름이 마이너스이고 개선이 보이지 않는다
- [ ] 경영진에 정직성 오점이 있다
- [ ] 경쟁우위가 돌이킬 수 없게 침식되고 있다
- [ ] "다음 인수자가 더 높은 가격을 낸다"에 기대 돈을 벌어야 한다(폭탄 돌리기)
- [ ] 이 투자가 0이 되는 결과를 감당할 수 없다
- [ ] 매수 이유가 주로 "남들이 다 산다" 또는 "최근 많이 올랐다"이다
- [ ] 매수 이유를 200자 이내로 명확히 쓸 수 없다

---

### 6단계: 총괄 비교표 출력(복수 회사 시 반드시 출력)

복수 회사를 분석할 때는 반드시 비교 총괄표를 생성한다:

| 회사 | Checklist 통과? | 능력범위 | 좋은 사업 | 해자 | 경영진 | 안전마진 | 핵심 결론 |
|------|----------------|--------|--------|--------|--------|---------|---------|
| | | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | |

---

### 7단계: 최종 결론 및 파일 작성

각 회사에 대해 명확한 결론을 제시한다(회피하지 않음):
- ✅ **Checklist 통과**(X/6관) —— 심층 연구 단계로 진입 가능
- ❌ **Checklist 미통과** —— 어느 레드라인이 발동됐는지 설명
- ❓ **회색 지대** —— 핵심 쟁점이 무엇인지, 투자자가 스스로 무엇을 판단해야 하는지 설명
- N/A —— 비상장/매수 불가

완전한 보고서를 `~/버핏Checklist-[회사명 또는 "다공사비교"].md`에 작성한다

## 출력 형식 요건

1. 각 회사를 독립된 장으로 구성, 다음 포함: 6관문 점수표 + 핵심 데이터표 + 핵심 리스크(3-5개) + 거울 테스트 + 명확한 결론
2. 복수 회사 시 마지막에 총괄 비교표 첨부
3. 모든 점수는 반드시 ★ 기호 사용(★1-5), 반쪽 별 미포함
4. 데이터는 반드시 출처 시점을 표기, 추정값은 반드시 "추정"으로 명시
5. 글 말미에 결어를 붙여, 버핏의 명언으로 호응: "투자의 제1원칙은 돈을 잃지 않는 것이다"
6. 언어 스타일: 직접적, 날카롭게, 군더더기 없이. 버핏/멍거/돤융핑(段永平)의 어록을 곁들여 평한다

## 핵심 원칙

- **놓칠지언정, 틀리지 말라**: Checklist의 목표는 나쁜 선택을 배제하는 것이지, 최고를 찾는 것이 아니다
- **능력범위를 정직하게 마주하라**: 이해할 수 없으면 이해할 수 없다고 말하라, 억지로 분석하지 말라
- **안전마진은 생명선**: 좋은 회사도 비싸게 사면 돈을 잃는다
- **거울 테스트는 건너뛸 수 없다**: 이유를 명확히 말할 수 없으면 사지 않는다, 예외 없다
