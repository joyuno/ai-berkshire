# 포트폴리오 관리: "기업 리서치"에서 "포트폴리오 관리"로

$ARGUMENTS 에 대해 투자 포트폴리오 점검과 최적화를 수행한다.

**지원 입력 형식**:
- 보유 목록, 예: `텐센트 30%, 메이퇀 20%, 마오타이 20%, 엔비디아 15%, 현금 15%`
- 또는: `텐센트 500주 @480홍콩달러, 메이퇀 1000주 @130홍콩달러, ...`
- 또는: `내 보유 종목`(저장된 포트폴리오 파일 `reports/portfolio-latest.md` 가 이미 있는 경우)

> "분산 투자는 무지에 대한 보호다. 자기가 무엇을 하는지 안다면 분산 투자는 의미가 없다." —— Warren Buffett(버핏)
>
> "내 평생 본 진짜 좋은 투자 기회는 열 손가락으로 셀 수 있다." —— 李录(리루)

## 설계 철학

기업 리서치는 투자의 절반에 불과하다. 나머지 절반은 **포트폴리오 차원의 의사결정**이다:
- 얼마나 살 것인가?(포지션)
- 어떤 돈으로 살 것인가?(자금 출처——신규 자금인가 종목 교체인가)
- 기존 보유와 충돌하는가?(상관관계)
- 최적 포트폴리오는 어떤 모습인가?(기회비용)

Warren Buffett(버핏)은 한 종목을 고립적으로 보지 않는다——그는 항상 "이것이 내가 할 수 있는 최선인가?"를 생각한다.

## 실행 프로세스

### 첫째 단계: 보유 종목 파싱

입력에서 현재 보유 종목을 파싱하여 다음 형식으로 표준화한다:

| 종목 | 코드 | 보유량 | 매입가 | 현재가 | 시가총액 | 비중 | 손익 |
|------|------|--------|-------|------|------|------|------|

입력에 비율만 있고 금액이 없으면 비율로 분석하면 된다.

동시에 기존 포트폴리오 파일(`reports/portfolio-latest.md`)이 있는지 확인하고, 있으면 읽어서 업데이트한다.

### 둘째 단계: 최신 데이터 획득

Task 도구로 백그라운드 Agent를 띄워 WebSearch로 각 보유 종목에 대해 병렬로 다음을 획득한다:
1. 현재 주가와 밸류에이션 지표(PE, PB, 배당수익률)
2. 가장 최근 분기의 핵심 재무 변화
3. 최근의 중대 사건
4. 애널리스트 컨센서스(선행 PE, 목표 주가)

각 보유 종목에 대해 `tools/financial_rigor.py verify-valuation` 으로 밸류에이션 데이터를 검증한다. 각 보유 종목에 정보 풍부도(A/B/C 등급)를 표기하고, C 등급 종목의 분석 결론에는 낮은 신뢰도를 표기한다.

### 셋째 단계: 개별 포지션 건강검진

각 보유 종목에 대해 빠른 건강 점검을 수행한다:

| 종목 | 현재 PE | 매수 논리 변화 여부 | 투자 논거 건강도 | 포지션 제안 |
|------|:------:|:--------------:|:---------:|---------|
| 텐센트 | 18x | 변화 없음 | 8/10 | 적정 |
| 메이퇀 | 25x | 경쟁 심화 | 6/10 | 다소 높음, 비중 축소 고려 |

각 보유 종목에 대해 답한다:
- [ ] **오늘 보유하고 있지 않다면, 현재 가격에 다시 매수하겠는가?**
- [ ] **내일부터 거래할 수 없다면, 5년 보유해도 편안한가?**
- [ ] **매수 논거가 아직 온전한가?**

**段永平(돤융핑)**: "어떤 주식을 10년 보유하고 싶지 않다면, 단 하루도 보유하지 마라."

### 넷째 단계: 포트폴리오 차원 분석

#### 4.1 집중도 분석

| 지표 | 현재값 | 권장 범위 | 판단 |
|------|-------|---------|------|
| 1위 보유 비중 | | <40% | |
| 상위 3개 보유 비중 | | 50~80% | |
| 총 보유 종목 수 | | 5~15개 | |
| 현금 비중 | | 10~30%(시장 환경에 따라) | |

**李录(리루)의 기준**: 핵심 보유 3~5개, 상위 3개가 80%+. **단, 이는 종목 하나하나를 철저히 리서치할 것을 요구한다.**

**Warren Buffett(버핏)의 기준**: 핵심 보유는 10개를 넘지 않되, 더 많은 위성 포지션은 허용한다.

#### 4.2 상관관계 점검

보유 종목 간의 숨은 연관성을 식별한다:

| 보유 A | 보유 B | 상관 유형 | 리스크 |
|-------|-------|---------|------|
| 텐센트 | 콰이쇼우 | 모두 중국 인터넷 | 규제 리스크 공진 |
| 엔비디아 | TSMC | AI 공급망 상하류 | AI Capex 동향 변동 |
| 메이퇀 | 핀둬둬 | 모두 중국 소비 | 거시 소비 동향 변동 |

**점검 목록**:
- [ ] 50%를 넘는 포지션이 같은 테마/산업에 노출되어 있는가?
- [ ] 50%를 넘는 포지션이 같은 국가/통화에 노출되어 있는가?
- [ ] 미중 관계가 악화되면 포트폴리오는 얼마나 손실이 나는가?
- [ ] 글로벌 경기 침체가 오면 포트폴리오는 얼마나 손실이 나는가?

#### 4.3 기회비용 분석

이것이 Warren Buffett(버핏)의 가장 핵심적인 사고방식이다——**모든 1원은 수익이 가장 높은 곳에 두어야 한다**.

모든 보유 종목을 "기대 연환산 수익"으로 정렬한다:

| 순위 | 종목 | 현재 비중 | 기대 연환산 수익 | 확실성 | 기대수익×확실성 |
|:----:|------|:-------:|:----------:|:------:|:--------------:|
| 1 | | | | | |
| 2 | | | | | |
| ... | | | | | |

기대수익 추정 방법(`tools/financial_rigor.py three-scenario` 로 계산):
- **간이 공식**: 기대 연환산 ≈ FCF Yield + 기대 성장률(주요 방법)
- **가치형 검증**: 안전마진 회귀 + 이익 성장률 + 배당수익률
- **성장형 검증**: 이익 성장률 × 적정 PE의 변화

**핵심 질문**: 최하위 보유 종목의 기대수익이 현금(무위험 이자율 ~4%)보다 높은가? 아니라면 매도하여 현금으로 바꿔야 한다.

#### 4.4 스트레스 테스트

| 시나리오 | 가정 | 포트폴리오 예상 영향 | 최대 낙폭 |
|------|------|-----------|---------|
| 글로벌 침체 | 기업 이익 20~30% 하락 | | |
| 미중 갈등 격화 | 중국 개념주 50% 할인 | | |
| 금리 급등 | 10년물 국채 → 6% | | |
| 기술 버블 붕괴 | 기술주 PE 40% 압축 | | |

각 시나리오에 대해 정성+개략 추정 평가를 한다(각 보유 종목의 산업 속성과 과거 밸류에이션 변동 범위에 기반):
- 어느 보유 종목이 가장 큰 충격을 받는가? 대략적 영향 방향과 크기 범위
- 포트폴리오 전체가 견딜 수 있는가?
- 헤지가 필요한가?

| 비중 확대 | | | | |
| 비중 축소 | | | | |
| 전량 매도 | | | | |
| 신규 매수 | | | | |
| 유지 | | | | |

#### 5.2 대체 종목 탐색

포트폴리오에 "현금만 못한" 포지션이 있거나 현금 비중이 과도하면, 본 Skill 안에서 직접 개별 종목을 추천하기보다 `/industry-research` 나 `/investment-checklist` 를 사용해 관심 산업/기업을 체계적으로 선별할 것을 권장한다.

#### 5.3 현금 관리

| 현재 현금 비중 | 제안 현금 비중 | 이유 |
|:----------:|:----------:|------|

**Warren Buffett(버핏)**: 현재 3,820억 달러의 현금을 보유, 총자산의 25% 초과——좋은 기회를 찾을 수 없을 때 현금이 최선의 포지션이다.

### 여섯째 단계: 포트폴리오 보고서 출력

#### 보고서 구조

```
1. 포트폴리오 개요(보유 표 + 원형 차트 설명)
2. 개별 포지션 건강검진(각 보유 종목의 건강 상태)
3. 포트폴리오 분석
   - 집중도: 과도하게 분산/집중되어 있는가?
   - 상관관계: 숨은 연관과 리스크 공진
   - 기회비용: 최하위 포지션을 보유할 가치가 있는가?
   - 스트레스 테스트: 극단 시나리오에서의 낙폭 추정
4. 리밸런싱 제안(구체적 행동 + 이유)
5. 다음 점검 시점과 주목 포인트
```

#### 결론은 반드시 명확히 답한다

1. **포트폴리오 전체 건강도**: 우수 / 양호 / 조정 필요 / 심각한 문제
2. **가장 먼저 해야 할 한 가지는 무엇인가?**(X 비중 확대 / Y 비중 축소 / 유지)
3. **현재 가장 큰 리스크는 무엇인가?**

### 일곱째 단계: 포트폴리오 파일 저장

포트폴리오 정보를 `reports/portfolio-latest.md` 에 기록하며, 다음을 포함한다:
- 최신 보유 표
- 이번 점검 일자와 결론
- 리밸런싱 기록(추가 누적)
- 다음 점검 알림

## 핵심 원칙

- **모든 1원에는 기회비용이 있다** — 평범한 주식을 보유하는 비용은 우수한 주식을 놓치는 것이다
- **집중이 리스크가 아니라 무지가 리스크다** — 깊이 이해하는 3종목 보유가 어설프게 아는 30종목 보유보다 안전하다
- **현금도 하나의 포지션이다** — 좋은 기회를 못 찾을 때 현금을 보유하는 것은 부끄러운 일이 아니다
- **포트폴리오 차원 > 개별 종목 차원** — 좋은 주식도 잘못된 포지션에서는 발목을 잡는다
- **정기적으로 점검하되 과도하게 거래하지 않는다** — 분기마다 한 번 점검이면 충분하고, 매일 시세를 보며 리밸런싱하지 않는다
