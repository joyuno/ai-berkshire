# 산업 투자 연구: 산업 사슬 전체 조망 스캔 + 4대 거장 개별 종목 분석 프레임워크

$ARGUMENTS 산업에 대해 체계적인 산업 사슬 투자 연구를 수행한다.

## 연구 목표

하나의 투자 테마/논리 사슬에서 출발하여 다음을 완성한다:
1. 투자 논리 사슬의 각 단계를 검증
2. 완전한 산업 사슬 전체 조망도 작성
3. 전 세계 모든 상장사 스캔(A주/홍콩주/미국주/국제)
4. 각 세부 단계의 선두 회사에 대해 4대 거장 프레임워크 분석 수행
5. 산업급 포트폴리오 배분 제안 출력

---

## 1단계: 투자 논리 사슬 구축과 검증

### 1.1 논리 사슬 그리기
"기저 트렌드"에서 "수혜 종목"까지의 인과 관계를 화살표 사슬로 표현한다, 예:
```
기저 트렌드 A
    → 수요 B를 유발
        → 병목/필수 수요 C를 창출
            → 수혜 산업 사슬 D
```

### 1.2 단계별 검증
논리 사슬의 각 화살표에 대해 의문을 제기하고 증거를 찾는다:

| 단계 | 핵심 가정 | 검증 방식 | 데이터 출처 |
|------|---------|---------|---------|
| A→B | | 산업 데이터/전망 검색 | |
| B→C | | 수급 분석 검색 | |
| C→D | | 실제 사례/계약 검색 | |

### 1.3 "이미 일어난 검증 이벤트" 찾기
해당 논리 사슬을 뒷받침하는 **이미 체결/실현된 실제 비즈니스 이벤트**(전망이 아니라)를 나열한다, 예: 대기업의 구매 계약, 정책 문서, 산업 보고서 등.

---

## 2단계: 산업 사슬 전체 조망도 작성

### 2.1 산업 사슬 구조 작성
산업을 상류→중류→하류→보조 단계로 분해한다, 예:
```
상류: 원자재/자원 채굴 → 소재 가공/정제
중류: 핵심 장비 제조 → 시스템 통합/엔지니어링 시공 → 신기술 연구개발
하류: 운영/서비스 → 최종 고객
보조: 검사/인증 → 유지보수 서비스 → 금융 도구(ETF/신탁)
```

### 2.2 각 단계의 "사업 특성" 식별
각 단계에 대해 표기한다:

| 단계 | 비즈니스 모델 | 매출총이익률 구간 | 경쟁 구도 | 장벽 유형 | 주기성 |
|------|---------|-----------|---------|---------|--------|
| | 자원 판매/장비 판매/서비스 판매/임대료 수취 | | 독점/과점/완전경쟁 | 자원/라이선스/기술/규모 | 강/중/약 |

### 2.3 "병목 단계" 표시
산업 사슬에서 공급이 가장 빠듯하고, 대체가 가장 어려우며, 이익률이 가장 높은 단계를 식별한다 — 이런 단계가 최적의 투자 종목이 있는 곳인 경우가 많다.

---

## AI 연구 편향 자각: 산업 연구의 특수한 함정

산업 연구에서는 AI 데이터 편향이 독특한 방식으로 증폭된다:

**산업급 편향**:
| 편향 유형 | 양상 | 대응 |
|---------|------|------|
| 성숙 산업 선호 | 전통 산업(은행/에너지/소비)은 자료가 매우 많아 AI 분석이 "더 확실해" 보인다 | 확실성은 비즈니스 모델에서 오는 것이지 리서치 자료 수에서 오는 것이 아니다 |
| 신흥 산업 과소평가 | 신산업(AI 응용/합성생물 등)은 자료가 적어 AI 분석이 보수적으로 치우친다 | "현재 데이터"가 아니라 "종착점 사고"로 산업 가치를 판단 |
| 대장주 선호 | 대기업 자료가 소기업보다 훨씬 많아, AI가 천성적으로 대장주를 추천하는 경향 | 소기업이 더 좋은 위험 대비 수익률을 가질 수 있으니, AI 분석 분량이 짧다고 무시하지 말 것 |
| 상장 선호 | 상장사만 스캔하면 산업 사슬의 핵심 비상장 플레이어를 놓친다 | 반드시 비상장 회사를 검색하고 "향후 IPO 후보"로 표기 |
| 영문 선호 | AI는 영문 자료 처리 능력이 더 강해, 중국/아시아 시장 플레이어를 과소평가할 수 있다 | 반드시 중·영문 정보원을 동시에 검색 |

**산업 사슬 스캔에서의 반(反)편향 조치**:
1. 각 단계에 대해 "AI가 쉽게 찾는 회사"만 나열하지 말고, "비주류이지만 우량할 수 있는 종목"을 능동적으로 검색
2. 정보가 희소한 소형주에 대해 분석 분량이 짧다고 추천도를 낮추지 말 것 — 보고서 길이가 아니라 핵심 질문(사업 본질, 해자, 경영진)으로 평가
3. 최종 보고서에서 각 회사의 "정보 충분도"(A/B/C등급)를 표기하여, AI 분석의 신뢰 정도를 독자가 알 수 있게 할 것

## 3단계: 전 세계 상장사 스캔

Task 도구로 백그라운드 Agent를 띄워, 해당 산업의 모든 상장사를 전면 검색한다.

### 검색 목록
- 미국주(NYSE/NASDAQ/NYSE American) 관련 회사
- 한국주(KOSPI/KOSDAQ) 관련 회사 — `py tools/krx_data.py search {키워드}`
- A주(상하이/선전) 관련 회사
- 홍콩주 관련 회사
- 기타 국제 시장(일본/유럽/호주 등)
- 산업 ETF
- 핵심 비상장 회사(향후 IPO 가능성)

### 회사별 수집 항목
- 회사명(중·영문)
- 종목 코드와 거래소
- 시총(근사)
- 한 줄 설명(산업 사슬에서의 위치와 역할)
- 순수 종목 여부(순수 원전 vs 다각화 사업 중 원전 사업 보유)
- 소속 산업 사슬 단계

### 출력 형식
산업 사슬 단계별로 분류하여, 각 단계마다 표 하나에 스캔된 모든 회사를 포함한다.
다시 투자 확실성에 따라 계층화한다:
- **Tier 1**: 대형 시총, 순수 종목, 업종 대장주
- **Tier 2**: 중형 시총, 순수하거나 높은 비중, 세부 분야 대장주
- **Tier 3**: 소형 시총, 개발 단계, 고위험·고변동성
- **Tier 4**: 다각화 기업 중 관련 사업을 보유한 대형 기업

---

## 4단계: 각 단계 선두 회사 4대 거장 분석

각 산업 사슬 단계의 **Tier 1·Tier 2 회사**에 대해 다음 분석을 수행한다(Tier 3/4 회사는 간단한 코멘트로 충분):

### 4.1 사업 본질(돤융핑(段永平))
- 이 회사가 산업 사슬에서 무엇을 하는지 한 줄로 정의
- 매출 구조와 성장률
- 매출총이익률/순이익률 수준 및 추세
- 현금흐름 특성
- **추궁**: 이것은 좋은 사업인가? 왜 그런가?

### 4.2 해자(버핏)
5종 해자로 점수를 매긴다(★1-5):

| 해자 | 강도 | 증거 |
|--------|------|------|
| 브랜드/가격결정력 | | |
| 전환비용 | | |
| 네트워크 효과 | | |
| 규모의 경제 | | |
| 기술/라이선스 장벽 | | |

**추궁**: 10년 후에도 해자가 남아 있는가?

### 4.3 리스크(멍거)
- 이 회사가 가장 실패하기 쉬운 방식은?
- 최악의 시나리오에서 가치는 얼마인가?
- 똑똑한 사람들은 왜 이 회사를 사지 않는가?

### 4.4 경영진(돤융핑(段永平) + 버핏)
- CEO/창업자는 누구인가? 핵심 의사결정 기록
- 지분율과 이해관계 일치
- 간단 평가(A/B/C등급)

### 4.5 밸류에이션 스냅샷
- 현재 PE/PS/EV/EBITDA
- 동일 단계 경쟁사와 대비
- 간단 평가: 비쌈/합리적/쌈

### 4.6 추천도
★1-5로 표기:
- ★★★★★ = 핵심 비중 후보
- ★★★★☆ = 위성 비중 후보
- ★★★☆☆ = 관찰 목록
- ★★☆☆☆ = 고위험 옵션
- ★☆☆☆☆ = 추천하지 않음

## 5단계: 산업급 리스크 평가(멍거 "체크리스트")

### 5.1 시스템 리스크 목록

| 리스크 | 확률 | 영향 | 대응 전략 |
|------|------|------|---------|
| 투자 논리 사슬의 어느 단계가 반증됨 | | | |
| 대체 기술 출현 | | | |
| 정책/규제 블랙스완 | | | |
| 수요의 주기적 조정 | | | |
| 밸류에이션 버블 붕괴 | | | |

### 5.2 역사적 비유
역사상 유사한 산업 사슬 투자 테마를 찾아, 그 최종 결말을 분석한다:
- 비유 대상 산업은 무엇인가?
- 최종 승자는 누구였는가?(상류/중류/하류?)
- 대다수 투자자는 돈을 벌었는가, 잃었는가?
- 현재 산업에 주는 시사점은 무엇인가?

### 5.3 편향 자가 점검
- 서사 편향: 스토리가 너무 완벽하지 않은가?
- 앵커링 효과: 최근 상승률에 닻을 내리고 있지는 않은가?
- 군중 추종: "모두가 사고 있어서" 사는 것은 아닌가?

## 6단계: 문명 트렌드 판단(리루(李录) 프레임워크)

- 이 산업이 의존하는 기저 트렌드는 "문명급 패러다임 전환"인가, "일시적 열풍"인가?
- 역사상 가장 가까운 기술 혁명 비유는 무엇인가?
- 10-20년 후, 이 산업의 종착점은 무엇인가?
- 산업 사슬에서 어느 단계가 "승자독식"이 될 가능성이 가장 높은가?
- 어느 단계가 가장 파괴(disruption)되기 쉬운가?

## 출력 요구사항

1. 모든 분석은 데이터 근거가 있어야 하며, 데이터 출처를 첨부
2. 핵심 데이터는 Markdown 표로 제시
3. 산업 사슬 전체 조망도는 코드 블록의 텍스트 도표로 표현
4. 각 단계마다 최소 2-3개 선두 회사를 분석
5. 전 세계 회사 스캔은 가능한 한 완전하게(A주/홍콩주/미국주/국제)
6. 최종적으로 완전한 보고서를 `~/[산업명]산업사슬투자연구보고서.md`에 작성
7. 결론은 명확하게, 구체적인 종목·비중·가격 구간 제안을 제시
8. 각 분석 모듈 말미에 해당 거장의 "추궁"을 둘 것

## 데이터 샘플 검증(출고 프로세스)

보고서 작성 후, 데이터 샘플 검증을 실행하고 통과해야 발행한다:

```bash
# Step 1 — 샘플 검증 목록 추출(15% 무작위 표본)
python3 ~/ai-berkshire/tools/report_audit.py extract \
  --report <보고서 파일 경로>

# Step 2 — 목록의 각 항목을 신뢰할 수 있는 정보원에서 취득(skills/financial-data.md 참조)

# Step 3 — 출고/반려 판정 출력
python3 ~/ai-berkshire/tools/report_audit.py verdict \
  --results '<채워 넣은 JSON>' \
  --report <보고서 파일명>
```

**【출고】** 전부 통과 → 보고서 발행 가능; **【반려】** 불합격 항목 있음 → 수정 후 재심사.
