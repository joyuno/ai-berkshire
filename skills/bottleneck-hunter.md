# 공급망 병목 헌터: AI 기반 글로벌 산업망 병목 차익거래

$ARGUMENTS 슈퍼 트렌드에 대해 공급망 병목 스캔과 차익거래 기회 발굴을 수행한다.

## 핵심 철학

"AI가 어떤 종목을 추천하느냐"를 묻지 말고, "이 트렌드가 계속 확장된다면 어느 환절이 먼저 부족해지는가?"를 물어라.

전통적 투자 리서치는 선두 기업과 이미 알려진 트랙을 좇는다. 이 시스템은 그 반대다. **물리적 공급망의 목줄(咽喉) 위치에서 출발해, 아무도 주목하지 않지만 일단 공급이 끊기면 업계 전체가 멈춰서 기다려야 하는 회사를 찾는다.**

초과수익의 원천: 1차 병목(GPU, HBM, 전력)은 이미 충분히 가격에 반영되어 있다. 진짜 알파는 **2차, 3차 병목**에 있다. 광모듈, 레이저, InP 기판, SOI 웨이퍼, 에피택시 장비, 웨이퍼 레벨 테스트, IC 기판, 특수 유리섬유 등이다.

## 1단계: 슈퍼 트렌드 확인

### 1.1 트렌드 선별 기준

작은 바람몰이 속에서 환상을 찾지 말고, 다음 조건을 전부 충족하는 슈퍼 트렌드만 좇는다.

| 기준 | 요구 사항 | 검증 방법 |
|------|------|---------|
| 지속성 | 최소 3~5년 확실한 성장 | 산업 전망, 자본지출 계획 검색 |
| 물리성 | 실제 하드웨어/소재/장비 구축이 필요 | "소프트웨어 업그레이드"와 "물리적 증설"을 구분 |
| 규모성 | 글로벌 자본지출 연 500억 달러 초과 | 선두 플레이어의 capex 가이던스 검색 |
| 가속성 | 수요 증가율 > 공급 증설 속도 | 수요 성장률 vs 생산능력 확장 계획 비교 |

### 1.2 현재 추적 중인 슈퍼 트렌드 목록

매번 실행할 때마다 갱신한다. 초기 목록:

1. **AI 인프라 구축** — 데이터센터, GPU 클러스터, 네트워크 상호연결, 전력
2. **에너지 전환** — 원전 재가동, 전력망 업그레이드, 에너지 저장
3. **국방 현대화** — 서방의 군비 상승 사이클, 공급망 재편
4. **반도체 재산업화** — 미국·유럽·일본의 보조금 기반 공장 건설, 장비/소재 병목
5. **우주 경제** — 위성 인터넷, 발사 빈도 급증

사용자가 구체적 트렌드(예: "AI 인프라")를 지정하면 해당 트렌드에만 집중한다.

### 1.3 트렌드 검증 출력

```
트렌드 명칭：
핵심 동력：（한 문장）
이미 발생한 검증 이벤트（최소 3개）：
  1. [날짜] [이벤트] [출처]
  2.
  3.
자본지출 규모：글로벌 약 $XX억/년, 증가율 YY%
수급 격차 판단：수요 증가율 > 공급 증설 속도？예/아니오/불확실
트렌드 확인：✅ 추적 가능 / ❌ 증거 부족, 당분간 추적 안 함
```

## 2단계: 공급망 물리적 분해

### 2.1 계층 분해 프레임워크

**개념 층에 머물지 말고 물리적 실체까지 분해한다.**

```
Layer 0（최종단）：최종 제품/서비스
    │
Layer 1（핵심 부품）：이미 시장이 충분히 주목한 핵심 하드웨어
    │                 ⬆ 가격 반영 충분, 알파 제한적
    │─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
    │                 ⬇ 관심도 낮음, 알파 집중 구간
    │
Layer 2（하위 부품/소재）：핵심 부품을 떠받치는 부품과 소재
    │
Layer 3（상류 장비/원료）：하위 부품 제조에 필요한 장비와 원자재
    │
Layer 4（인프라）：전력, 냉각, 토지, 인재, 인증
```

### 2.2 AI 인프라를 예로 든 분해 템플릿

```
Layer 0：AI 모델 학습/추론 서비스
Layer 1：GPU/가속기, HBM 메모리, 서버, 데이터센터
Layer 2（중점 스캔 구간）：
  ├─ 네트워크 상호연결：광모듈, 광섬유, 스위치 칩, 동축 케이블
  ├─ 광통신 코어：레이저（EML/VCSEL/CW）, 변조기, 광검출기
  ├─ 반도체 소재：InP 기판, GaAs 기판, SOI 웨이퍼, SiC 기판
  ├─ 첨단 패키징：CoWoS 기판, HBM TSV, ABF 기판막
  ├─ PCB/기판：고주파 고속 PCB, IC 기판, 특수 유리섬유 직물
  ├─ 테스트：웨이퍼 레벨 테스트（Probe Card）, 번인 테스트, ATE
  ├─ 방열/냉각：액랭 시스템, CDU, 침지식 냉각액
  └─ 전력 연결：버스덕트, UPS, 배전반, 변압기
Layer 3：
  ├─ 에피택시 장비：MOCVD, MBE
  ├─ 노광/식각：특수 파장 노광, InP 식각
  ├─ 원자재：고순도 금속（인듐, 갈륨, 게르마늄）, 특수가스, 타겟재
  └─ 인증/표준：MSA 표준, Telcordia 인증
Layer 4：
  ├─ 전력：원전, 천연가스 발전, 송변전
  ├─ 냉각수/방열 인프라
  └─ 데이터센터 토지/인허가
```

## 3단계: 병목 식별 — "목줄" 찾기

### 3.1 병목 판정 6대 기준

Layer 2~3의 각 환절에 대해 항목별로 평가한다.

| # | 기준 | 질문 | 점수 |
|---|------|------|------|
| 1 | **공급 집중도** | 글로벌 공급사 3곳 이하? | 🔴 2곳 이하 / 🟡 3~5곳 / 🟢 5곳 초과 |
| 2 | **증설 주기** | 신규 생산능력 확보에 얼마나 걸리는가? | 🔴 2년 초과 / 🟡 1~2년 / 🟢 1년 미만 |
| 3 | **대체 난이도** | 다른 기술/소재로 대체 가능한가? | 🔴 대체 불가 / 🟡 부분 대체 가능 / 🟢 대체 용이 |
| 4 | **가동률** | 현재 생산능력 가동률은? | 🔴 90% 초과 / 🟡 70~90% / 🟢 70% 미만 |
| 5 | **수요 증가율** | 하류 수요 증가율은? | 🔴 연 50% 초과 / 🟡 20~50% / 🟢 20% 미만 |
| 6 | **고객 검증 주기** | 신규 공급사 진입에 얼마나 검증이 필요한가? | 🔴 1년 초과 / 🟡 6~12개월 / 🟢 6개월 미만 |

**병목 등급**:
- 🔴🔴🔴 4개 이상 → **S급 병목**（단일 장애점 수준, 최우선）
- 🔴🔴 3개 → **A급 병목**（심각하게 제약됨）
- 🔴 1~2개 → **B급 병목**（압력은 있으나 통제 가능）
- 🔴 없음 → 병목 아님, 건너뜀

### 3.2 병목 지도 출력

```
공급망 병목 지도 — {트렌드 명칭}
갱신일：YYYY-MM-DD

S급 병목（단일 장애점）：
  1. [환절명] — [원인 한 문장] — 공급사：[회사 목록]
  2.

A급 병목（심각하게 제약）：
  1.
  2.

B급 병목（압력 있음）：
  1.
  2.

최근 변화（지난 스캔 대비）：
  - [신규/등급상향/등급하향/해소] [환절명] — [원인]
```

## 4단계: 기업 선별 — 병목에서 종목으로

### 4.1 각 S급·A급 병목에 대해, 관련 상장사를 모두 찾아낸다

검색 방식:
- WebSearch "{병목 환절} supplier listed company"
- WebSearch "{병목 환절} manufacturer stock"
- WebSearch "{병목 제품} market share company"

### 4.2 1차 선별 기준（빠른 필터링）

| 기준 | 요구 사항 | 근거 |
|------|------|------|
| 상장 여부 | 이미 상장（A주/홍콩/미국/일본/대만/유럽） | 거래 가능 |
| 병목 사업 비중 | 매출의 30% 초과가 병목 환절에서 발생 | 순도 |
| 시가총액 | 100억 달러 미만 우선 | 대형주는 이미 충분히 가격 반영됨 |
| 유동성 | 일평균 거래대금 100만 달러 초과 | 진입·청산 가능 |

### 4.2.1 밸류에이션 점검（반드시 수행, 건너뛸 수 없음）

**병목이 진짜라고 해서 투자 기회인 것은 아니다.** 각 회사에 대해 PS, PE를 반드시 계산하고 보고서에 명기한다. 아래 복합 조건으로 밸류에이션이 과도하게 선반영되었는지 판단한다.

#### 밸류에이션 적신호（아래 중 하나라도 충족 → 신호 강도 최대 ★★로 제한, "⚠️ 밸류에이션 과열" 표기）

1. **시가총액 > TAM의 20%**: 회사 시총이 도달 가능 시장 규모의 20%를 이미 초과 — 성장 기대가 과도하게 내재화되었음을 의미
2. **PS > 30x 이면서 매출 증가율 < 100%**: 고밸류에이션이지만 증가율이 이를 뒷받침하기에 부족. 증가율 100% 초과 기업은 PS 적신호에서 면제되나, 여전히 "⚠️ 고밸류에이션, 고성장 지속 검증 필요" 표기 필요
3. **시가총액 > 5년 낙관적 매출 전망의 10배**: 가장 낙관적인 가정이 전부 실현되어도 현재 가격이 여전히 과도함
4. **증자 후 60일 내 주가 2배**: 심리 주도 특성이 뚜렷, 신호 강도 1등급 하향

#### 밸류에이션 황색신호（추가 설명 필요, 없으면 강등）

1. **적자 + PS > 15x**: ★★★ 진입은 허용하되 흑자 전환 경로와 일정을 반드시 설명
2. **PS가 동종 흑자 기업의 5배 이상**: 프리미엄의 근거（시장 점유율, 성장률 차이, 진입장벽 차이）를 반드시 설명
3. **PE > 80x**: PEG를 계산하고 증가율이 이를 뒷받침하는지 설명 필요

#### 밸류에이션 청색신호（가점 항목）

- PS < 10x 이면서 매출 성장 중 → 신호 강도 1등급 가산 가능
- PE < 30x 이면서 해자 보유 → "밸류에이션에 안전마진 있음" 표기

#### 밸류에이션 합리성 검증（필수）

각 종목에 대해 답한다: "현재 시가총액에 매수하고, 가장 낙관적 시나리오가 전부 실현되어 10년 후 25x PE로 청산한다고 가정하면 연환산 수익률은 얼마인가?" 연환산 수익률 10% 미만 → "현재 가격은 안전마진이 없음" 표기.

**유의**: 밸류에이션 점검의 목적은 "PS 100배의 적자 기업"을 추천하는 명백한 오류를 막는 것이지, 모든 고밸류에이션 초기 기업을 배제하는 것이 아니다. 핵심은 증가율, TAM, 경쟁 구도가 현재 밸류에이션을 뒷받침할 수 있는지이며, 일률적 적용이 아니라 구체적 분석이 필요하다.

### 4.3 심층 선별 차원

1차 선별을 통과한 기업에 대해 한 곳씩 평가한다.

```
## {회사명}（{코드}）

**병목 포지셔닝**：
- 공급망에서의 구체적 위치
- 시장 점유율：글로벌 X위, 비중 XX%
- 고객 명단（알려진 것）

**생산능력과 확장**：
- 현재 생산능력 / 가동률
- 증설 계획 / 일정
- 증설 소요 자금 vs 보유 현금

**재무 스냅샷**：
- 시가총액 / 매출 / 이익 / 증가율
- 병목 사업 매출 비중
- 매출총이익률 추이（병목이 조일수록 매출총이익률은 상승해야 함）

**리스크 체크리스트**：
- [ ] 대체 기술 리스크：우회될 수 있는가?
- [ ] 희석 리스크：대규모 증자/전환사채가 있는가?
- [ ] 지정학 리스크：민감 지역에 있거나 수출 통제 대상인가?
- [ ] 경영진 리스크：부정적 이력이 있는가?
- [ ] 고객 집중 리스크：단일 고객에 과도하게 의존하는가?
- [ ] 밸류에이션 과열：현재 밸류에이션이 이미 3년치 성장을 반영했는가?

**병목 지속성 판단**：
- 이 병목은 언제 해소될 것인가?
- 해소된 후 이 회사에 무엇이 남는가?
- 일회성인가 지속적인가?
```

## 5단계: 교차 검증 — 한 가지 스토리만 듣지 않는다

### 5.1 정방향 검증

| 검증 항목 | 질문 | 검색 방식 |
|--------|------|---------|
| 고객 검증 | 주요 고객이 이미 계약/도입했는가? | 회사 공시, 고객 실적 발표 언급 검색 |
| 매출 검증 | 병목이 이미 매출 성장에 반영되었는가? | 최근 2~3개 분기 실적 검색 |
| 가격 검증 | 제품 가격이 오르고 있는가? | 업계 호가, 애널리스트 리포트 검색 |
| 생산능력 검증 | 생산능력이 정말 빡빡한가? | 납기 데이터, 고객 불만 검색 |
| 자본 검증 | 증설 자본지출이 있는가? | 회사 capex 가이던스 검색 |

### 5.2 역방향 검증（멍거식 부정）

| 역방향 질문 | 의미 |
|---------|------|
| 똑똑한 사람들은 왜 이 종목을 사지 않는가? | 알려진 약세 논거 탐색 |
| 이 병목은 우회될 수 있는가? 어떤 대체 경로가 있는가? | 기술 경로 리스크 |
| 중국/기타 플레이어가 생산능력을 빠르게 복제할 수 있는가? | 공급 충격 리스크 |
| 최종 수요가 50% 둔화되면 이 회사는 어떻게 되는가? | 하방 민감도 |
| 경영진이 과거 고점에서 증자로 희석한 적이 있는가? | 경영진 신뢰도 |
| 현재 밸류에이션은 어떤 성장 가정을 내포하는가? | 밸류에이션 합리성 |

### 5.3 신호 교차 검증

- 같은 병목에 속한 여러 회사가 모두 오르고 있는가?（업계 검증）
- 하류 고객이 실적 발표에서 공급 부족을 언급했는가?（고객 검증）
- 업계 협회/연구기관에 관련 데이터가 있는가?（제3자 검증）

### 6.1 병목 기회 순위표

|------|------|------|------|--------|-----|-----|---------|---------|---------|---------|---------|---------|
| 1 | | | | | x | x | | S/A | | | ★1-5 | 합리/다소 높음/과열 |

**필수 입력 항목**: 시가총액, 연매출, PS, PE는 필수이며 "확인 필요"로 건너뛸 수 없다. 재무 데이터를 확보할 수 없으면 신호 강도는 ★★를 초과할 수 없다.

신호 강도 등급（밸류에이션 점검 결과가 등급에 직접 영향）:
- ★★★★★ 다중 교차 검증, 고객 이미 도입, 매출에 이미 반영, 밸류에이션 청색신호（PS 합리 + 흑자 또는 흑자 근접）
- ★★★★ 대부분 검증 통과, 밸류에이션 청색 또는 황색신호（설명 첨부 필요）
- ★★★ 논리는 성립하나 일부 미검증, 밸류에이션 황색신호 수용 가능（예: 고성장 초기 기업）
- ★★ 초기 신호, 또는 병목 논리는 성립하나 밸류에이션 적신호（시총 > TAM 20%, PS > 30x 이면서 증가율 부족, 시총이 5년 전망을 크게 초과 등）
- ★ 순수 콘셉트, 미검증

### 7.1 매 실행 시의 증분 갱신

2. 새로 출현한 병목 스캔
   - 최근 7일의 supply chain / shortage / bottleneck 뉴스 검색
   - 실적 시즌 중 공급망 관련 disclosure 점검

3. 병목 등급 갱신（상향/하향/해소）

## 매시간 스캔 모드（정기 작업용）

매시간 1회 실행하며 "건질 게 있을 때만 보고서를 낸다"는 방식을 채택한다.

### 스캔 절차（매시간）

1. **뉴스 스캔**: 지난 1~2시간의 공급망 관련 뉴스 검색
   - 키워드：supply chain bottleneck, shortage, capacity constraint, allocation, lead time, sole source, 병목, 품귀, 생산능력, 가격 인상
   - 커버리지：영문 + 중문 소스
2. **시장 신호**: 추적 중인 회사의 주가 변화 점검（특히 이상 변동 5% 초과 주목）
3. **실적/공시**: 병목 관련 회사의 실적 발표나 중대 공시 여부 점검
4. **밸류에이션 기회**: 관찰 명단의 회사가 시장 전반 하락 등으로 매수 구간에 진입했는지 점검
5. **보고서 여부 판단**:
   - 새 병목 신호, 명확한 종목 기회, 중대한 상태 변화가 있으면 → **보고서 발행**
   - 새로운 발견이 없으면 → **보고서 미발행**, 로그에 "이번 라운드 신규 신호 없음"만 기록

### 보고서 출력 규칙

**하루에 폴더 하나**: `reports/bottleneck-map/YYYY-MM-DD/`

**파일 명명 규칙**（파일명만 봐도 종목이 있는지 한눈에 파악）:

| 상황 | 파일명 형식 | 예시 |

| 병목 신호는 있으나 명확한 종목 없음 | `HH-MM-신호스캔.md` | `14-00-신호스캔.md` |

## AI 리서치 편향 자각

| 편향 | 양상 | 대응 |
|------|-----|------|
| 선두주 선호 | 검색 결과가 대형주에 점령됨 | 의도적으로 소형 공급사 검색, "small cap" 키워드 추가 |
| 영어 선호 | 일본·한국·대만 기업 누락 | 일본/한국/대만 시장의 공급사를 반드시 검색 |
| 내러티브 선호 | "AI 콘셉트" 라벨에 끌림 | 실제 공급망 위치만 보고 시장 라벨은 보지 않음 |
| 확증 편향 | 병목을 찾은 뒤 긍정적 증거만 찾음 | 역방향 검증을 강제 수행（5단계） |

## 핵심 원칙（최고 우선순위）

1. **AI에게 종목을 추천하게 하지 말고, 공급망을 분해하게 하라** — 답보다 질문이 중요하다
2. **물리 우선** — 실제 물리적 제품/소재/장비가 필요한 환절에만 주목한다
3. **2차, 3차** — 이미 충분히 가격에 반영된 선두주를 좇지 않는다
4. **교차 검증** — 모든 결론은 독립적 정보원 최소 2개
5. **불확실성에 정직하라** — 데이터를 못 찾으면 "데이터 부족"이라 쓰고, 추측으로 채우지 않는다
6. **병목에는 시효가 있다** — 모든 병목은 결국 해소된다, 관건은 시간 창을 판단하는 것
7. **소형주 ≠ 좋은 기회** — 소형주도 형편없는 회사일 수 있으니 재무 품질 관문을 반드시 통과시킨다
8. **병목이 진짜 ≠ 투자 기회** — 한 회사가 가장 빡빡한 병목에 앉아 있더라도, PS > 30x이거나 여전히 적자라면 현재 가격은 매수 시점이 아니다. **밸류에이션은 절대 문턱이며, 병목 순도, 신호 강도, 내러티브 매력으로 덮을 수 없다.** 오른 병목주를 놓칠지언정, PS 100배에 적자 회사를 매수하지 마라
9. **CLAUDE.md 객관성 원칙 준수** — 강세를 미리 전제하지 말고, 데이터 먼저 결론 나중
