# 경영진 심층 연구: 주식을 산다는 것은 사람을 사는 것

$ARGUMENTS 에 대해 경영진 심층 연구를 수행한다.

**지원 입력 형식**: `회사명` 또는 `인물명 회사명`, 예: `메이퇀`, `왕싱 메이퇀`, `젠슨 황 엔비디아`

> "주식을 산다는 것은 곧 사람을 사는 것이다. 신뢰할 수 있는 사람을 찾아, 오래 보유하라." —— 段永平(돤융핑)
>
> "경영진을 평가하려면, 아무도 보지 않을 때 그들이 무엇을 하는지를 보라." —— Warren Buffett(버핏)

## 설계 철학

대부분의 투자 분석은 경영진 평가를 표면에서 멈춘다: 경력, 지분율, 보수. 그러나 Warren Buffett(버핏)은 **경영진과 식사하며 대화하는 데** 많은 시간을 쓰고, 李录(리루)는 **투자의 본질은 사람에 대한 투자**라고 말하며, 段永平(돤융핑)은 **주식을 산다는 것은 사람을 사는 것**이라고 말한다.

본 Skill은 `/investment-research` 5단계 경영진 평가의 **심화판**이다. 표준 투자 연구에서 경영진 점수가 불확실하거나(★★★ 이하), 경영진이 핵심 투자 논리일 때 본 Skill로 심층 연구를 한다.

AI는 경영진과 식사할 수 없지만, 공개 정보를 통해 다음을 해낼 수 있다:
- **경영진의 말과 행동이 일치하는지 추적**(약속 vs 이행)
- **모든 중대한 자본 배분 결정의 수익률 분석**
- **어려운 시기의 결정에서 인품 추론**
- **직원/입점업체/고객의 피드백으로 측면 검증**

## 실행 흐름

### 1단계: 핵심 경영진 식별 및 병렬 데이터 수집 시작

WebSearch로 다음 핵심 인물을 확인한다:

| 역할 | 성명 | 재임 기간 | 배경 | 지분/옵션 |
|------|------|------|------|----------|
| CEO/이사회 의장 | | | | |
| CFO | | | | |
| 창업자(현직 아닌 경우) | | | | |
| 실질 지배자(CEO와 다른 경우) | | | | |
| 기타 핵심 임원 | | | | |

**유의**: "누가 의사결정을 하는가"와 "누구의 이름이 직함에 있는가"를 구분하라. 일부 회사는 창업자가 물러났어도 여전히 영혼과 같은 존재다(예: 핀둬둬의 황정).

핵심 인물 확인 후, Task 도구로 여러 백그라운드 Agent를 **병렬** 가동해 다음 데이터를 수집한다:
1. Agent 1: CEO의 공개 발언과 예측 기록(주주서한, 컨퍼런스콜, 인터뷰, 소셜미디어)
2. Agent 2: 자본 배분 결정 기록(M&A, 자사주 매입, 배당, 신규 사업 투자)
3. Agent 3: 지배구조와 보수(지분 구조, 특수관계인 거래, 임원 보수)
4. Agent 4: 측면 검증 정보(직원 평가, 고객 피드백, 업계 평판)

### 2단계: CEO 능력범위 평가

#### 2.1 전략적 안목

CEO의 지난 5년간 공개 발언(주주서한, 컨퍼런스콜, 인터뷰, 소셜미디어)을 검색해, 다음 질문에 대한 그의 판단을 추출한다:

| 시기 | CEO의 판단/예측 | 실제 결과 | 정확도 |
|------|--------------|---------|:------:|
| | "우리는 X 시장이 ...할 것으로 본다" | X 시장 실제로... | ✅/❌ |
| | "향후 3년 우리의 중점은..." | 실제 실행... | ✅/❌ |

**핵심 질문**:
- CEO가 시장을 앞선 올바른 판단을 한 적이 있는가?
- CEO가 모두가 낙관할 때 냉정함을 유지한 적이 있는가?
- CEO의 산업 트렌드 이해는 시장을 따라가는 것인가, 독립적 사고인가?

#### 2.2 실행력

| 차원 | 평가 | 증거 |
|------|------|------|
| 전략의 실현 | 말한 것을 해냈는가? | |
| 조직력 | 인재를 유치하고 붙잡을 수 있는가? | |
| 위기 대응 | 어려움에 부딪혔을 때 어떻게 대처하는가? | |
| 반복 개선 속도 | 실수 후 바로잡는 속도가 빠른가? | |

### 3단계: 정직성 평가(가장 중요)

**Warren Buffett(버핏)**: "우리는 세 가지 자질을 찾는다: 정직, 지성, 그리고 에너지. 첫 번째가 없으면, 나머지 둘은 당신을 망친다."

#### 3.1 약속 vs 이행 추적

지난 3년간의 실적 컨퍼런스콜, 주주서한, 공개 인터뷰에서 경영진이 한 **구체적 약속**을 추출한다:

| # | 시기 | 약속 내용 | 약속 자리 | 이행 상황 | 평가 |
|---|------|---------|---------|---------|------|
| 1 | | "2025년 X 사업 흑자 전환을 실현하겠다" | 2024년 연간실적 컨퍼런스콜 | | ✅/⚠️/❌ |
| 2 | | "X억 규모 자사주 매입을 계획한다" | 2024년 주주서한 | | ✅/⚠️/❌ |

**이행률 통계**:

| 약속 이행률 | 평가 |
|:---------:|------|
| >80% | 우수 —— 말한 대로 한다 |
| 60-80% | 합격 —— 큰 방향은 맞으나 실행에 편차 |
| 40-60% | 우려 —— 약속이 과하고 이행이 부족 |
| <40% | 심각한 문제 —— 신뢰할 수 없음 |

#### 3.2 어려운 시기의 행동

회사 역사상 겪은 중대한 위기/어려움(주가 폭락, 실적 미스, 규제 충격, 경쟁 심화)을 검색해, 경영진의 대응을 분석한다:

| 위기 사건 | 시기 | 경영진 반응 | 사후 돌아본 평가 |
|---------|------|-----------|-------------|

**주목점**:
- 능동적으로 소통하는가, 회피하는가?
- 원인을 내부에서 찾는가, 외부에 떠넘기는가?
- 기회로 삼아 어렵지만 옳은 일을 하는가, 단기적으로 시장에 영합하는가?

#### 3.3 이해관계자에 대한 태도

| 이해관계자 | 경영진 태도 | 증거 | 평가 |
|-----------|-----------|------|------|
| 주주 | 존중/무시/이용 | | |
| 직원 | 우대/착취/방치 | | |
| 고객/사용자 | 고객 중심/단기 착취 | | |
| 입점업체/공급업체 | 공정 협력/극단적 단가 압박 | | |
| 규제/사회 | 규정 준수 협조/편법 | | |

**李录(리루)**: "이해관계자에 대한 태도가 기업의 장기 생명력을 결정한다. 단기 착취는 효율을 높이지만, 장기적으로는 생태계를 해친다."

### 4단계: 자본 배분 능력

이것이 Warren Buffett(버핏)이 가장 중시하는 경영진 능력이다 —— **1원을 벌 때마다, 경영진은 그것을 얼마로 바꿀 수 있는가?**

#### 4.1 자본 배분 결정 기록

회사의 지난 5년간 중대한 자본 배분 결정을 검색해, 건별로 평가한다:

**M&A 기록**:

| 시기 | 인수 대상 | 금액 | 전략적 논리 | 사후 수익 | 점수(1-5) |
|------|---------|------|---------|---------|:---------:|

**자사주 매입 기록**:

`tools/financial_rigor.py verify-valuation`을 사용해 매입 당시와 현재의 PE 등 밸류에이션 지표를 검증한다.

| 시기 | 매입 금액 | 평균 매입가 | 당시 PE | 사후 돌아보기 | 점수(1-5) |
|------|---------|-----------|:------:|---------|:---------:|

**배당 기록**:

| 연도 | 배당 금액 | 배당성향 | 동기 FCF | 지속 가능 여부 |
|------|---------|:------:|---------|:---------:|

**신규 사업 투자**:

| 시기 | 투자 분야 | 누적 투입 | 현재 상태 | 수익 평가 | 점수(1-5) |
|------|---------|---------|---------|---------|:---------:|

#### 4.2 자본 배분 점수

| 차원 | 점수(1-5) | 설명 |
|------|:---------:|------|
| M&A 규율 | | 합리적 가격에 인수하는가? 인수 후 통합은 어떤가? |
| 자사주 매입 타이밍 | | 저평가 시 매입하고 고평가 시 멈추는가? |
| 배당의 합리성 | | 배당성향이 FCF와 부합하는가? |
| 신규 사업 투자 | | 성공률은? 손절 규율은? |
| 현금 관리 | | 현금 보유가 합리적인가? 과도하게 쌓아두지 않는가? |
| **종합 점수** | | |

**Warren Buffett(버핏) 기준**: 이상적인 경영진은 좋은 기회가 있을 때 과감히 투자하고, 좋은 기회가 없을 때 적극적으로 자사주 매입/배당을 하며, 결코 비싼 가격에 인수하지 않는다.

### 8단계: 경영진 평가 보고서 출력

```
1. 핵심 인물 한눈에 보기(표)

3. 능력 평가
```

|------|:----:|:---------:|:----:|
| 정직성 | 35% | | |
| 전략 및 실행력 | 25% | | |
| 자본 배분 능력 | 25% | | |

| **종합 점수** | 100% | | |

#### 段永平(돤융핑)의 "사람을 산다" 기준

> 다음 세 가지 질문에 답하라:
> 1. **이 사람은 정직한가?**(정직하고, 주주의 이익을 가로채지 않는가)
> 2. **이 사람은 능력이 있는가?**(전략적 안목 + 실행력 + 자본 배분)
> 3. **당신은 이 사람에게 돈을 맡겨 10년간 운용하게 하겠는가?**
>
> 셋 다 "예" = ★★★★★(5점)
> 앞의 둘이 "예" = ★★★★(4점)
> 첫 번째만 "예" = ★★★(3점)
> 첫 번째가 "예"가 아니다 = ★(1점, 투자하지 않음)

## 핵심 원칙

- **정직성은 한 표 거부권 항목** —— 능력 부족은 배울 수 있지만, 인품의 문제는 고칠 수 없다
- **말이 아닌 행동을 보라** —— 경영진이 무엇을 말하는지는 중요하지 않다, 무엇을 했는지가 중요하다
- **어려움 속에서 진실을 본다** —— 순풍일 때는 누구나 좋은 CEO다, 역풍일 때라야 진짜 실력이 드러난다
- **자본 배분은 최종 시험** —— 돈 버는 것은 쉽다, 번 돈을 잘 배분하는 것이 어렵다
- **경영진과 사랑에 빠지지 마라** —— 객관성을 유지하라, 당신이 흠모하는 사람도 큰 실수를 할 수 있다
