# 암호화폐 리서치: 메인 코인 펀더멘털·온체인·지정학 분석

$ARGUMENTS 메인 코인(BTC·ETH 등)에 대해 기관급 펀더멘털 리서치를 수행한다.

## 적용 범위와 한계 (먼저 읽을 것)

- **대상**: 메인 코인(BTC, ETH) 위주. **알트코인 long tail은 제외**한다(요청 시 별도 분석).
- **제외 1 — 기술적 분석(TA)**: 차트 매매 시그널·진입가·목표가는 **다루지 않는다**. 이 도구는 리서치 하네스이지 트레이딩 데스크가 아니다. 지표 "수치"는 인용하되 매매 판단은 하지 않는다.
- **제외 2 — 트레이딩 콜**: "사라/팔아라" 식 단정 콜은 책임질 수 없어 출력하지 않는다. 대신 시나리오·확률·조건을 제시한다.
- **프레임 주의**: 본 프로젝트의 4대가 가치투자 프레임(현금흐름·해자)은 코인에 **직접 적용 불가**하다. 따라서 코인 네이티브 평가틀을 쓰되, 4대가 회의론은 **반대편 검증 장치(steelman)**로만 사용한다.

## 핵심 원칙 (최고 우선순위)

- **객관·객관·객관**: 사실(온체인 데이터)과 관점(내러티브)을 엄격히 구분한다.
- **입장 미리 정하지 않기**: 코인 맥시멀리즘도, 무조건 회의론도 아니다. 데이터 → 논리 → 결론 순.
- **정·반 양면 병기**: 모든 핵심 판단에 강세론과 약세론을 함께 제시한다.
- **추정치 명시**: 불확실하면 "추정"·"불확실/데이터부족"으로 표기. 추측으로 확실성을 위장하지 않는다.
- **LLM 암산 금지**: 시세·시총·공급은 `tools/crypto_data.py`로 정량 취득.

## 1단계: 자산 분류와 평가틀 선택

코인마다 본질이 달라 평가틀이 다르다. 먼저 분류한다.

| 유형 | 대표 | 핵심 질문 | 평가틀 |
|------|------|----------|--------|
| **가치저장형(SoV)** | BTC | "디지털 금이 될 것인가?" | 희소성·채택률·금 시총 대비·보유분포·네트워크 보안 |
| **L1 플랫폼형(생산자산)** | ETH | "디지털 경제 인프라로 수수료를 버는가?" | 수수료 매출·소각/발행·스테이킹 수익·생태계 TVL·경쟁 점유율 |

> BTC는 "현금흐름이 없다"는 비판이 **본질적으로 맞다** → 가치저장 내러티브의 채택률로 평가.
> ETH는 수수료·소각이라는 **유사 현금흐름**이 있어 생산자산 틀(P/F, 실질수익률)을 부분 적용 가능.

## 4단계: 회의론 검증 (steelman — 약점을 가장 강하게 변호)

반편향 장치. 강세 논리에 도취되지 않도록 **실패 시나리오를 먼저 강제**한다.

- **Warren Buffett(버핏)·Charlie Munger(멍거)식**: "현금흐름 없는 투기 자산" 비판 — 어디까지 유효하고 어디서 반박 가능한가?
- **段永平(돤융핑)식**: "이해 못 하면 안 한다" — 일반 보유자가 이걸 이해하고 10년 보유할 수 있는가?
- **구조적 실패 경로(각 코인 3가지)**: 규제 압살 / 스테이블코인發 시스템 리스크 / 채굴·스테이킹 중앙화 / 양자컴퓨팅 / 커스터디·거래소 리스크 / 무한 경쟁 가치희석
- **"이번엔 다르다"의 함정**: 과거 사이클(2018·2022) 대비 구조적으로 다른 점 vs 똑같은 점
- **90% 폭락·무의미화 경로**: 각 코인이 영구 손상될 수 있는 경로를 구체적으로 적는다.

## 5단계: 밸류에이션·시나리오 (TA 아님 — 펀더멘털 시나리오)

매매 콜이 아니라 **조건부 시나리오**로 제시한다.

### 가치저장형(BTC)
- 금 시총($X조) 대비 BTC가 Y% 흡수 시 함의 시총 → 함의 가격(추정)
- 채택률 시나리오: 강세(기관·국가 가속) / 기준 / 약세(규제·내러티브 붕괴)

### L1 플랫폼형(ETH)
- 수수료 매출 기반 유사 밸류에이션(P/F, 실질수익률), 순공급 증감
- 시나리오: L2 확장 성공 / 정체 / 경쟁 L1에 점유율 상실

```
시나리오 표 (코인별):
| 시나리오 | 전제 | 함의 시총/가격(추정) | 확률(정성) |
| 강세    |      |                    |          |
| 기준    |      |                    |          |
| 약세    |      |                    |          |
```

> 모든 가격은 **추정**이며 목표가가 아니다. "현재 시점 펀더멘털이 함의하는 범위"일 뿐.
