# 산업 깔때기 선별: 전체 시장에서 3개 종목까지 좁히는 가치투자 정선 프로세스

$ARGUMENTS 산업/방향에 대해 깔때기식 가치투자 선별을 수행하여, 전체 시장 스캔에서 단계별로 정선해 최종 3개 종목까지 좁힌다.

## 적용 상황

특정 산업이나 투자 방향(예: "AI 컴퓨팅 파워", "혁신 신약", "로봇")을 제시하며 다음을 원할 때:
1. 중요한 종목을 하나도 놓치지 않기(A주, 홍콩주, 미국주, 비상장 후보 포함)
2. 통일된 기준으로 "스토리주"와 품질이 부족한 회사를 걸러내기
3. 진짜 깊이 연구할 가치가 있는 상위 3개 종목에 역량 집중하기
4. 각 단계마다 명확한 잔류/탈락 기준을 두어 복기·추적 가능하게 하기

`industry-research`와의 차이:
- `industry-research`는 산업 사슬 구조와 전체 조망에 무게를 두며, 단계(환절)별로 잘라 본다
- `industry-funnel`은 개별 종목 선별 깔때기에 무게를 두며, 전체 시장에서 단계별로 3개까지 정선한다

둘은 상호 보완적이다: 먼저 `industry-research`로 산업 사슬 구도를 파악하고, 이어서 `industry-funnel`로 종목을 정선한다.

## 깔때기 구조 개요

```
1단계: 전체 시장 스캔     30-60개   （활성도 + 상승률 + 시총 상위 30의 합집합）
        ↓ 가치투자 5개 핵심 지표
2단계: 1차 선별           ≤ 10개    （5개 항목 전부 합격 + 해자 ★★★ 이상）
        ↓ 정밀 분석
3단계: 정밀 분석          ≤ 10개    （각 회사 300-500자 구조화 분석）
        ↓ 최종 선정
4단계: 4대 거장 심층 분석  3개       （각 회사 800-1200자, Warren Buffett(버핏)·Charlie Munger(멍거)·段永平(돤융핑)·李录(리루) 관점）
        ↓
출력: 투자 의견 + 매매 신호 + 비중 제안
```

각 단계에서 "걸러낸 종목"은 반드시 탈락 사유를 남겨야 하며, 블랙박스가 되어서는 안 된다.

## 1단계: 전체 시장 스캔 진입

### 1.1 활성 종목 정의(세 부류의 합집합)

**A 부류 - 거래 활성도**:
- 최근 30일 일평균 거래대금 기준 업종 상위권(A주/홍콩주/미국주/한국주 각각 상위 30개)

**B 부류 - 상승률 순위**:
- 최근 30일 상승률 상위 20개
- 최근 90일 상승률 상위 20개
- 둘의 합집합

**C 부류 - 시총 기준**:
- 업종 내 시총 상위 30개(등락 무관)

최종 스캔 풀 = A ∪ B ∪ C, 예상 30-60개.

### 1.2 반드시 검색해야 할 시장

| 시장 | 출처 제안 |
|------|---------|
| A주(상하이·선전) | 통화순/동방재부 업종 섹터, 통다신 |
| 홍콩주 | 푸투/통화순 홍콩주, HKEX 업종 분류 |
| 미국주 | NASDAQ/NYSE 업종 ETF 보유 종목, Yahoo Finance |
| 한국주(KOSPI/KOSDAQ) | 네이버 금융 업종/테마, KRX 업종분류, `py tools/krx_data.py search {키워드}` |
| 국제 시장 | 일본·대만·유럽의 관련 회사를 놓치지 말 것(특히 반도체, 전자) |
| 비상장 회사 | "향후 IPO 후보" 별도 소항목으로 정리하고, 최신 밸류에이션과 잠재 IPO 시점을 명시 |

### 1.3 출력 형식

| 회사명 | 코드 | 시장 | 시총 | 주력 사업 한 줄 | 해당 산업 비중 | 선정 부류(A/B/C) |
|-------|------|-----|------|----------|-----------|----------------|

**핵심 자가 점검**:
- 산업 비중 < 30%인 "연관주"는 신중하게 다루고 "순수 종목 아님"으로 표시
- 중국/아시아 시장은 영문 자료가 적다는 이유로 누락하지 말 것
- 소형주는 AI가 대형주를 선호한다는 이유로 누락하지 말 것

## 2단계: 가치투자 5개 핵심 지표 1차 선별 → ≤ 10개

1단계의 30-60개 회사에 대해, 한 곳씩 5개 핵심 지표를 적용한다.

### 2.1 5개 핵심 지표

| # | 지표 | 통과 기준 | 완화 조건 | 데이터 출처 |
|---|------|---------|---------|---------|
| 1 | PE 밸류에이션 | 합리적(역사 구간·동종업계 대비) | 고성장은 PEG < 1.5까지 완화 가능 | 재무제표 + wind/통화순 |
| 2 | ROE | > 15% 또는 최근 3년 추세 개선 | 자본집약 업종은 완화 가능 | 재무제표 |
| 3 | 영업현금흐름 | 양(+)이며 순이익의 > 70% | — | 재무제표 |
| 4 | 부채비율 | < 60% | 유틸리티/전력은 70%까지 완화 | 재무제표 |
| 5 | 해자 신속 평가 | ★★★ 이상 | — | 정성 판단 |

**해자 5종**:
- 브랜드/가격결정력
- 전환비용/사용자 고착성
- 네트워크 효과
- 규모의 경제
- 기술/라이선스/자원 장벽

### 2.2 출력 형식

| 회사 | PE | ROE | 현금흐름/순이익 | 부채비율 | 해자 | 종합 | 잔류/탈락 | 탈락 사유 |
|------|----|----|-----------|-------|-------|------|------|---------|

**잔류 규칙**:
- 5개 항목 전부 합격 → 즉시 잔류
- 4개 합격 + 1개 근접 → 잔류하되 노란색 표시
- 4개 미만 → 탈락, 사유 명시

**목표**: 잔류 ≤ 10개. 너무 많이 남으면(> 12개) 해자 기준을 ★★★★로 올려 한 번 더 선별한다.

## 3단계: 정밀 분석(≤ 10개, 각 회사 300-500자)

1차 선별에서 잔류한 회사에 대해 한 곳씩 구조화 분석을 한다.

### 3.1 회사별 분석 템플릿

```
## {회사명}（{코드}）

**비즈니스 모델 한 줄**:
（무엇을 파는가, 누구에게 파는가, 어떻게 돈을 버는가）

**재무 품질**:
- 매출 성장률 / 이익 성장률 / 매출총이익률 / ROE / 현금흐름
- 핵심 변화（최근 1-2년 가장 중요한 재무 전환점）

**해자 깊이**:
- 주요 해자 유형 + 구체적 증거
- 5년 후에도 해자가 남아 있을지에 대한 간략 판단

**주요 리스크（상위 3）**:
1.
2.
3.

**밸류에이션 신속 평가**:
- 현재 PE/PS/EV/EBITDA + 역사 구간 내 위치
- 동종업계 대비
- 한 줄 결론: 비쌈 / 합리적 / 쌈

**최종 3개에 진입?**: 예 / 아니오（사유）
```

### 3.2 최종 3개 선정 기준

점수 순위로 상위 3개를 뽑는 것이 아니라, "포트폴리오 상호 보완성"을 기준으로 선정한다:
- 최소 1개 "고확실성·저변동성"(Warren Buffett(버핏)형)
- 최소 1개 "중간 확실성·중간 변동성"(성장형)
- 선택적으로 1개 "고변동성·고위험"(옵션형)

특정 하위 트랙에서 충분히 좋은 3개를 찾을 수 없다면, 수를 억지로 채우지 말고 "최종 2개 + 관찰 1개"라고 적는 편이 낫다.

## 4단계: 4대 거장 심층 분석(3개, 각 회사 800-1200자)

최종 선정된 3개 회사에 대해 4대 거장 관점의 심층 분석을 수행한다.

### 4.1 段永平(돤융핑) 관점: 사업의 본질

- 이 회사가 무슨 사업을 하는지 한 줄로 정의
- 이것은 좋은 사업인가? 왜 그런가?
- 이 회사의 "본분(本分)"은 무엇인가? 경영진이 거기서 벗어나지는 않았는가?
- 비즈니스 모델의 "지속성"은 어디에 있는가?

### 4.2 Warren Buffett(버핏) 관점: 해자의 깊이

- 5종 해자로 점수 매기기(★1-5), 구체적 증거 제시
- 10년 후에도 해자가 남아 있는가?
- 지금 매수할 때의 "안전마진"은 어디에 있는가?

| 해자 | 강도 | 구체적 증거 |
|-------|------|---------|
| 브랜드/가격결정력 | | |
| 전환비용 | | |
| 네트워크 효과 | | |
| 규모의 경제 | | |
| 기술/라이선스 장벽 | | |

### 4.3 Charlie Munger(멍거) 관점: 리스크와 실패 모드

- 이 회사가 가장 실패하기 쉬운 방식은?(상위 3개 실패 경로 나열)
- 최악의 시나리오에서 가치는 얼마인가?(초간단 밸류에이션)
- 똑똑한 사람들은 왜 이 회사를 사지 않는가?(역방향 논증)
- 도덕적/컴플라이언스/경영진 리스크가 있는가?

### 4.4 李录(리루) 관점: 문명급 트렌드 내 위치

- 이 회사가 속한 트랙은 "문명급 패러다임 전환"인가, 아니면 "일시적 열풍"인가?
- 역사상 가장 가까운 기술 혁명 비유는?
- 10-20년 후 이 회사의 종착점은?
- 승자독식 구도인가?

### 4.5 종합 추천도

```
추천도：★★★★☆
비중 유형：핵심 / 위성 / 옵션 / 관찰
매수 제안 구간：현재가 / N% 조정 / 인내심을 갖고 대기
제안 보유 비율：해당 테마 비중 중 X%
핵심 모니터링 지표：（이 회사의 논리가 뒤집히는 신호는 무엇인가）
```

## 5단계: 종합 출력

보고서 말미에 통합한다:

### 5.1 최종 3개 포트폴리오 표

| 회사 | 유형 | 추천도 | 제안 비중 | 핵심 논리 | 핵심 리스크 |
|------|------|-------|---------|---------|---------|
| A | 핵심 | ★★★★★ | 50-60% | | |
| B | 위성 | ★★★★☆ | 25-35% | | |
| C | 옵션 | ★★★☆☆ | 5-15% | | |

### 5.2 산업급 ETF 대안

종목을 직접 고르고 싶지 않다면, 관련 ETF 1-3개(A주/홍콩주/미국주)를 나열한다.

### 5.3 산업 전체 위치 판단

- 산업 PE/PB 역사 분위
- 자금 흐름(북향 자금, ETF 설정·환매, 셀사이드 커버리지 밀도)
- 전체적으로 "초기/확장/성숙/쇠퇴" 중 어느 단계인지

### 5.4 정보 충분도 자가 평가(필수 기입)

| 항목 | 등급 | 설명 |
|-----|------|-----|
| 회사 재무 데이터 완전성 | A/B/C | |
| 밸류에이션 데이터 시의성 | A/B/C | |
| 산업 구도 판단 | A/B/C | |
| 경영진 정보 | A/B/C | |

A = 데이터 충분·신뢰 가능; B = 일부 누락되었으나 주요 결론에 영향 없음; C = 누락이 많아 결론에 신중 필요.

### 5.5 업데이트 대기 데이터 포인트

명확히 나열할 것: 어떤 데이터가 추정값인지, 어떤 데이터가 추후 확인이 필요한지, 어느 분기 실적을 중점적으로 추적해야 하는지.

### 5.6 자료 출처 목록

각 데이터/결론의 출처 링크를 분류하여 나열(재무제표, 리서치 자료, 뉴스, 산업 보고서).

## AI 연구 편향 자각(중요)

깔때기 선별 과정에서 AI가 빠지기 쉬운 함정:

| 편향 | 양상 | 대응 |
|------|-----|------|
| 대장주 선호 | 대형주는 자료가 많고 분석 분량이 길어 "더 좋아" 보인다 | 핵심 지표와 해자 점수로 평가하고, 보고서 분량으로 순위를 매기지 말 것 |
| 영문 선호 | 미국주는 자료가 풍부해 A주·홍콩주가 과소평가되기 쉽다 | 반드시 중·영문 모두 검색하고, A/H 회사를 놓치지 말 것 |
| 스토리 선호 | 높은 상승률 + 미디어 열기 = 더 좋아 보이는 "AI 컨셉주" | "AI 매출 비중" vs "AI 스토리 비중"을 구분하고, 실제 사업을 볼 것 |
| 현재 선호 | 현재 재무가 좋은 회사가 선정되기 쉬워, 전환기의 다크호스를 놓칠 수 있다 | 2단계 1차 선별에서 "추세 개선"을 완화 조건으로 허용 |
| 상장 선호 | 상장사만 보면 트랙 최고의 플레이어를 놓칠 수 있다 | 반드시 "향후 IPO 후보"를 나열하고, 밸류에이션과 시점 윈도를 표시 |

## 출력 요구사항

1. **보고서 위치**: `reports/{산업명}-funnel-{YYYYMMDD}.md`(산업 보고서는 reports/ 루트 디렉토리에 둔다)
2. **언어**: 한국어
3. **스타일**: 직접적이고 날카롭게, 군말 없이
4. **데이터**: 모든 데이터는 출처를 표시; 추정값은 "추정"으로 표기
5. **입장을 미리 정하지 않음**: 먼저 데이터 제시 → 논리 전개 → 결론 도출
6. **양면 제시**: 모든 핵심 판단에 반대 논거를 덧붙임
7. **각 단계 탈락 기록 유지**: 탈락한 회사도 이름 + 사유를 남길 것

보고서 작성 후, 데이터 샘플 검증을 실행하고 통과해야 발행한다:

```bash
# Step 3 — 출고/반려 판정 출력
python3 ~/ai-berkshire/tools/report_audit.py verdict \
  --results '<채워 넣은 JSON>' \
  --report <보고서 파일명>
```

## 후속 동작

깔때기로 최종 3개를 선정한 후, 각 회사에 대해 개별적으로 실행할 수 있다:
- `/investment-team` —— 완전한 4대 거장 병렬 심층 연구(독립 하위 디렉토리 + 5개 문서)
- `/investment-checklist` —— Warren Buffett(버핏) 매수 전 checklist를 체계적으로 한 번 점검
- `/management-deep-dive` —— 경영진 종심 연구

`industry-funnel`은 진입점이고, 후속 skill은 심층 발굴이다.
