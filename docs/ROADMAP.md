# ai-berkshire 로드맵 (한국어판)

> 본 저장소는 한국·미국 시장 중심의 한국어판 포크다. 데이터·보고서·skill이 모두 한국어로 동작한다.

## P0: 단기 (1-2개월)

### 데이터 소스 확장
- ✅ **한국(KOSPI/KOSDAQ) 데이터 도구 `tools/krx_data.py` 완료** (네이버 금융 API, 무외부의존)
- 미국 데이터 자동화: macrotrends/stockanalysis/SEC EDGAR 웹검색 기반 (현재 동작 중)
- A주(akshare/东方财富)·홍콩주 데이터 보조 도구 유지·보강 (선택)
- 기존 skill 변경 없이 데이터 계층만 확장

## P1: 중기 (3-6개월)

### HTML 보고서 출력
- Markdown 기반에 HTML 보고서 형식 추가
- 다크모드·내비게이션바·차트 시각화 지원
- 보고서 전파력과 가독성 향상

### 다단계 심층 모드
- `lite`: 5분 속판 — 밸류에이션 구간과 핵심 결론만 빠르게 제시
- `standard`: 현재 기본 모드 — 완전한 멀티에이전트 리서치
- `deep`: 교차검증·역사 유추를 더 추가한 기관급 심층

### 다종목 횡적 비교
- 2-4개 종목을 동일 차원으로 횡적 대결 지원
- 동종업계 기업 밸류에이션 벤치마킹
- 비교 매트릭스와 우선순위 추천 출력

## P2: 장기 (6개월+)

### 테스트 커버리지
- 핵심 도구(financial_rigor.py, krx_data.py 등) 단위 테스트 추가
- skill 출력 회귀 테스트 추가
- 반복 개선이 기존 기능을 깨지 않도록 보장

### 포트폴리오 수준 분석
- 보유 조합 건전성 평가
- 산업/지역 집중도 분석
- 상관관계 리스크 탐지
