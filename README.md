# AI Berkshire — AI 시대의 가치투자 리서치 프레임워크

> "Price is what you pay, value is what you get." — 워런 버핏
>
> AI로 투자 리서치의 깊이와 효율을 다시 정의한다.

**AI Berkshire** 는 [Claude Code](https://claude.ai/code) 기반의 투자 리서치 Skill 모음입니다. 워런 버핏, 찰리 멍거, 돤융핑(段永平), 리루(李录) 네 명의 가치투자 대가의 방법론을 체계화·구조화하여, AI Agent로 전문가 수준의 투자 리서치를 구현합니다.

한 사람 + Claude = 하나의 투자 리서치 팀.

이 프레임워크는 한국(KOSPI/KOSDAQ)·미국·홍콩·중국(A주) 등 글로벌 시장 전반에 적용되며, 모든 보고서는 **한국어로 출력**됩니다.

## 왜 AI에게 그냥 물어보면 안 되는가?

물론 Claude에게 직접 물어볼 수 있습니다. "삼성전자 살 만한지 분석해줘"라고 하면, "한편으로는… 다른 한편으로는…" 식의 균형 잡힌 분석을 받은 뒤 "투자에는 리스크가 있으니 본인이 판단하세요"로 끝나는 답을 얻게 됩니다.

**이런 분석은 맞는 말처럼 보이지만, 의사결정에 쓸 수가 없습니다.**

AI Berkshire가 푸는 문제는 "분석을 할 수 있느냐"가 아니라 **분석의 품질과 의사결정 규율**입니다. 핵심 차이는 다음과 같습니다.

### 1. 결론을 강제한다, 두루뭉술하게 넘어가지 않는다

AI에게 그냥 물으면 양쪽 다 만족시키는 "분석"을 받습니다. AI Berkshire는 **통과/불통과/회색지대**를 구체적 가격 구간과 전략별 권고와 함께 강제로 출력합니다.

> 일반 AI 답변: *"이 회사는 성장 잠재력이 있지만 경쟁 압력에도 직면해 있어 투자자는 이를 저울질해야 합니다…"*
>
> AI Berkshire 출력:

> | 전략 | 권고 | 가격 구간 |
> |------|------|---------|
> | 공격형 | 현재가에서 20% 진입 가능 | $95-105 |
> | 안정형 | 자사주 정책 명확화 후 진입 | $85-95 |
> | 보수형 | 10년 확실성 기준 미충족, 관망 | — |
>
> **거울 테스트**: 다섯 문장으로 다 설명 못 하면 = 매수 금지. 예외 없음.

### 2. 단일 분석이 아닌, 4대 대가 시각의 대결

"버핏 방식으로 한번 분석해줘"처럼 단순하지 않습니다. 네 시각은 **진짜 모순과 긴장**을 만들어 냅니다.

예시(중국 핀둬둬/PDD 분석, 과거 산출물 예시):
- **돤융핑**(비즈니스 모델): 좋은 사업, C2M 모델은 복제하기 어렵다 → 평점 3.7/5
- **버핏**(재무·밸류에이션): 현금 차감 PE 6.3배에 불과한 현금인출기 → 평점 4.4/5
- **멍거**(역발상): 해자가 생각보다 얕다, 더우인이 3년 만에 GMV 4조 위안 달성 → 평점 3.5/5
- **리루**(장기 확실성): 경영진 문화에 위험 요소, 10년 후 불확실 → 평점 2.0/5

**버핏은 "정말 싸다", 리루는 "불확실하면 안 산다"** — 이런 충돌이야말로 투자 의사결정의 실제 모습입니다. 단일 프롬프트로는 이런 다시각 대결을 만들 수 없고, 바로 이것이 맹점을 피하는 핵심입니다.

### 3. 구조화된 편향 방지 메커니즘

AI에게 가장 위험한 것은 틀린 답이 아니라, **맞아 보이지만 따져보면 무너지는** 답입니다. AI Berkshire는 프로세스에 여러 층의 "속지 않기" 장치를 내장했습니다.

| 메커니즘 | 해결하는 문제 | 예시 |
|------|------------|------|
| **정보 풍부도 등급(A/B/C)** | "자료가 많다 = 확실성이 높다"는 착각 방지 | 팝마트(泡泡玛特)는 B등급: 데이터 제한, 추정 지표에 신뢰도 표기 |
| **멍거식 역방향 검증** | 실패 시나리오를 강제로 사고 | "어떤 경우에 이 회사가 죽는가?" → 5대 시나리오와 확률 제시 |
| **빠른 부결 체크리스트** | 8개 레드라인 한 표 부결 | 경영진 정직성 결함 → 밸류에이션 아무리 싸도 즉시 부결 |
| **반(反)컨센서스 점검** | 시장과 같은 생각 회피 | "똑똑한 사람들은 왜 공매도하는가?" → 간과된 리스크 발견 |
| **여백의 원칙** | 차라리 "모른다"고 말함 | 데이터 부족 시 "회색지대" 표기, 추측으로 확실성을 위장하지 않음 |

### 4. 금융 데이터의 정밀성

LLM의 암산은 신뢰할 수 없습니다. PE 소수점 한 자리를 틀리거나 시가총액 단위를 원화와 달러로 혼동하면 잘못된 투자 결정으로 이어질 수 있습니다.

**실제 사례**: 텐센트 분석 시, 출처마다 시가총액 데이터가 "홍콩달러 억"과 "위안화 억" 두 단위로 섞여 있었습니다. AI Berkshire의 처리 방식:

```bash
# 시가총액 수동 검증: 주가 × 총주식수, 보고 데이터와 대조
python3 tools/financial_rigor.py verify-market-cap \
  --price 510 --shares 9.11e9 --reported 4.65e12 --currency HKD
# ✅ 검증 통과, 편차 0.08%에 불과
```

모든 계산은 Python `decimal.Decimal`(정밀 십진수)을 사용하며 `float`을 쓰지 않습니다. 핵심 데이터는 최소 2개 독립 출처로 교차 검증합니다.

### 5. 재현 가능한 리서치 프로세스

AI에게 그냥 물으면 매번 출력 형식·깊이·커버리지가 달라집니다. 오늘은 텐센트에 해자 점수를 매겼다가, 내일 메이퇀을 분석할 땐 그걸 빼먹습니다.

AI Berkshire는 보장합니다: **같은 입력 → 구조도 깊이도 일관된 출력.** 덕분에 가능한 일은:
- 7개 회사를 동일 기준으로 횡단 비교
- 같은 회사를 반년 뒤 다시 분석해 변화를 직접 대조
- 팀원 간 리서치 결과 정렬

> 실제 출력 — 7개 회사를 동일 기준 체크리스트로 선별:

### 6. 멀티 Agent 병렬 = 리서치 깊이의 배가

`/investment-team`은 4개의 독립 Agent를 띄워 **동시에** 한 회사를 연구합니다. 각 Agent는 스스로 웹을 검색하고, 데이터를 교차 검증하고, 독립적으로 결론을 냅니다. 하나의 프롬프트를 네 토막으로 자른 게 아니라, 4명의 "애널리스트"가 각자 완전한 리서치를 수행하고 Team Lead가 종합하는 구조입니다.

혼자 AI에게 물으면 컨텍스트 윈도가 하나입니다. 4개 Agent 병렬은 곧 4배의 검색량, 4배의 정보원, 4개의 독립 시각입니다.

```
┌─────────────────────────────────────────────┐
│              Team Lead (당신)                │
│         총괄 조율 · 종합 판단                │
├──────┬──────┬──────────┬───────────┤
│ Agent 1    │ Agent 2    │ Agent 3        │ Agent 4         │
│ 비즈니스모델│ 재무·밸류   │ 산업·경쟁      │ 리스크·경영진    │
│ 돤융핑 시각 │ 버핏 시각   │ 멍거 시각      │ 리루 시각        │
└──────┴──────┴──────────┴───────────┘
        ↓ 병렬 리서치, 실시간 진행 보고 ↓
              최종 종합 보고서
```

## 시장 데이터 소스

이 프레임워크는 시장별로 적합한 데이터 소스를 사용합니다.

### 한국 시장 (KOSPI / KOSDAQ)

신규 도구 `tools/krx_data.py`로 한국 종목 데이터를 자동 취득합니다. 네이버 금융 API 기반이며 **외부 의존성이 없습니다.**

```bash
# 시세 조회 (삼성전자)
py tools/krx_data.py quote 005930

# 재무 데이터 (카카오)
py tools/krx_data.py financials 035720

# 종목 검색 (에코프로)
py tools/krx_data.py search 에코프로
```

### 미국 시장

macrotrends / stockanalysis / SEC EDGAR 웹검색을 기반으로 데이터를 수집합니다.

### 홍콩·중국(A주) 시장

기존 도구(설계바이두/동방재부/슈에추 등) 및 웹검색 기반으로 데이터를 수집합니다.

> 어느 시장이든 모든 보고서는 **한국어로 출력**됩니다.

### 1. `/investment-research` — 4대 대가 종합 분석

가장 종합적인 단일 회사 심층 리서치 프레임워크. 7개 모듈을 순차 실행합니다.

```
데이터 수집 → 사업의 본질(돤융핑) → 해자(버핏) → 역발상(멍거)
    → 경영진 평가(돤융핑+버핏) → 문명 트렌드(리루) → 밸류에이션과 안전마진
```

**핵심 특징**:
- AI 리서치 편향 자각 메커니즘 (A/B/C 정보 풍부도 등급)
- 핵심 데이터 다출처 교차 검증 (시가총액 수동 검증, 최소 2개 독립 출처)
- 4대 대가의 "추궁"이 전편을 관통
- 3개 시나리오 밸류에이션(낙관/중립/비관) + 역산 DCF

**출력 예시 발췌**:

> #### 종합 의사결정 메모
>
> | 차원 | 결론 | 신뢰도 |
> |------|------|--------|
> | 사업 품질(돤융핑) | 탁월: 플랫폼형 사업, 양면 네트워크 효과, 한계비용 0 수렴 | ★★★★★ |
> | 해자(버핏) | 넓고 더 넓어지는 중: 네트워크 효과+전환비용+규모의 경제 삼중 중첩 | ★★★★☆ |
> | 경영진(돤융핑+버핏) | 우수: 창업자 주도, 자본 배분 규율 강함 | ★★★★☆ |
> | 최대 리스크(멍거) | 규제 정책 불확실성, 신사업 적자가 전체 이익을 잠식 | ★★★☆☆ |
> | 문명 트렌드(리루) | 디지털 소비 트렌드에 부합, 다만 "문명급 패러다임 전환"은 아님 | ★★★★☆ |
> | 밸류에이션(버핏+돤융핑) | 현재 PE 18배, 역사적 중앙값보다 다소 낮음, 일정한 안전마진 | ★★★★☆ |
>
> **돤융핑**: "이 사업의 본질은 소비자와 상인을 연결하고, 효율 향상에서 돈을 버는 것이다. 좋은 사업의 표식은: 사용자가 많을수록 상인이 많고, 상인이 많을수록 사용자가 많다. 플라이휠은 한번 돌기 시작하면 멈추기 어렵다."
>
> **멍거**: "거꾸로 생각하라 — 만약 이 회사가 내일 사라진다면 사용자와 상인은 어떻게 할까? 답이 '곧 대체재를 찾는다'면 해자가 충분히 깊지 않다. 답이 '생활이 매우 불편해진다'면 주목할 가치가 있다."

### 3. `/investment-checklist` — 버핏 매수 전 체크리스트

6개 관문 빠른 선별로 한 회사가 깊이 연구할 가치가 있는지 10분 내에 판단합니다.

```
제1관: 능력 범위 (내가 이해할 수 있는가?)
    ↓ 통과
제2관: 좋은 사업 (경제적 특성은?)
    ↓ 통과
제3관: 해자 (경쟁 우위가 깊은가?)
    ↓ 통과
제4관: 경영진 (신뢰할 만한가?)
    ↓ 통과
제5관: 안전마진 (가격이 싼가?)
    ↓ 통과
제6관: 의사결정 규율 (이성인가 FOMO인가?)
    ↓ 통과
   ✅ 거울 테스트
```

**다중 회사 비교 지원** — 한 번에 여러 종목 선별:

```
/investment-checklist 삼성전자, 엔비디아, 애플, SK하이닉스
```

**출력 예시 발췌**:

> #### 거울 테스트
>
> "나는 380 HKD에 텐센트를 매수한다, 왜냐하면:
> 1. 이 사업의 본질은 **소셜 네트워크+디지털 콘텐츠 플랫폼**이고, 나는 그것을 이해한다;
> 2. 그 해자는 **12억 사용자의 소셜 관계 사슬**이고, 더 넓어지고 있다;
> 3. 경영진 **Pony Ma는 겸손하고 실용적이며 자본 배분이 우수**해 신뢰할 만하다;
> 4. 현재 가격은 내재가치의 **8할** 수준으로 일정한 안전마진이 있다;
> 5. 설령 내가 틀려도 하방 리스크는 통제 가능하다, 왜냐하면 **장부상 순현금 2000억 초과, 게임 현금흐름 강력**하기 때문이다."
>
> ✅ 거울 테스트 통과
>
> **다섯 문장으로 다 설명 못 하면 = 매수 금지. 예외 없음.**

### 4. `/industry-research` — 밸류체인 전경 스캔

하나의 투자 테마에서 출발해 밸류체인 전경 연구를 완성합니다.

```
투자 논리 사슬 구축 → 밸류체인 전경도 → 글로벌 상장사 스캔
    → 각 단계 선두 회사 4대 대가 분석 → 포트폴리오 배분 권고
```

**출력 예시 발췌**:

> #### 투자 논리 사슬: 원자력
>
> 기저 트렌드: AI 데이터센터 전력 수요 폭발 + 탄소중립 목표
> → 결과: 안정적 청정 기저부하 전원 수요 급증
> → 창출: 원전 재가동/신규 건설/SMR의 확실한 수요
> → 수혜: 우라늄 광산 → 연료 가공 → 설비 제조 → 운영사
>
> #### 추천 포트폴리오
>
> | 계층 | 비중 | 종목 | 단계 | 핵심 논리 |
> |------|------|------|------|---------|
> | 코어 | 50% | 중국광핵(CGN)、Cameco | 운영+우라늄 | 확실성 최고 |
> | 위성 | 30% | 중국핵전、동방전기 | 운영+설비 | 국산화 수혜 |
> | 옵션 | 15% | NuScale、Nano Nuclear | SMR | 고위험 고탄력 |
> | ETF | 대체 | URA、URNM | 전체 사슬 | 간편 솔루션 |

### 5. `/industry-funnel` — 산업 퍼널 스크리닝

하나의 산업/방향에서 출발해 **전체 시장 → ≤10개 → 3개**로 층층이 정밀 선별합니다.

```
전체 시장 스캔 (활성도 + 상승률 + 시총 상위 30 합집합, 30-60개)
    ↓ 가치투자 5개 하드 지표
1차 선별 ≤ 10개
    ↓ 정밀 분석 (각 300-500자)
정밀 분석 ≤ 10개
    ↓ 최종 선정 (포트폴리오 상호보완성 기준, 점수 상위 3 아님)
4대 대가 심층 분석 3개 (각 800-1200자)
    ↓
추천 포트폴리오 (코어 / 위성 / 옵션) + 매매 시그널
```

**핵심 특징**:
- 각 층마다 명확한 유지/탈락 기준, 탈락 종목은 탈락 사유를 남김 (블랙박스 아님)
- 최종 3개는 "포트폴리오 상호보완성"으로 선정 (고확실성 + 중탄력 + 고탄력), 점수 상위 3 정렬이 아님
- "미래 IPO 후보"를 강제로 나열해 1차 시장 핵심 플레이어 누락 방지
- AI 편향 자각 메커니즘: 선두주 선호 / 영어 선호 / 스토리 선호 / 상장사 선호에 대응

**`/industry-research`와의 차이**:
- `industry-research`는 밸류체인 구조와 전경에 무게 (단계별 슬라이스)
- `industry-funnel`은 개별주 선별 퍼널에 무게 (전체 시장에서 층층이 3개까지 정밀 선별)

**실측: AI 산업 4개 세부 트랙 병렬 (2026-05-09)**:

| 세부 트랙 | 최종 3개 | 코어 비중 추천 |
|-------|---------|------------|
| AI 컴퓨팅 | TSMC / NVIDIA / SK Hynix | TSMC ★★★★★ |
| AI 모델 | Alphabet / Meta / 알리바바 | Alphabet ★★★★★ |
| AI 응용 | 마이크로소프트 / Adobe / AppLovin | 마이크로소프트 + Adobe ★★★★ |
| AI 인프라 전력 | Eaton / 특변전공 / Talen Energy | Eaton + 특변전공 ★★★★ |

**핵심 발견**: AI 응용 계층의 최대 승자는 AI Native 회사가 아니라, 채널+데이터+워크플로우 임베드를 갖춘 성숙한 거인이다 — 이는 1995-2000 인터넷 버블의 "곡괭이 파는 자가 이긴다"는 역사 법칙과 호응한다 (아마존과 애플이 이기고, Pets.com이 진다).

전체 보고서: [AI 컴퓨팅](reports/AI算力-funnel-20260509.md) · [AI 모델](reports/AI模型-funnel-20260509.md) · [AI 응용](reports/AI应用-funnel-20260509.md) · [AI 인프라 전력](reports/AI基建电力-funnel-20260509.md)

### 7. `/news-pulse` — 주가 급변 뉴스 귀인

"주가 급등/급락 시 무슨 일인지 빠르게 파악"하기 위해 설계한 인텔리전스 대응 Skill. **심층 투자 리서치가 아니라 10-15분 빠른 귀인** — 보유 종목이 급변할 때 소설 쓰기 불안이나 맹목적 손절을 피합니다.

**핵심 차별화**:
- **4차원 병렬 정찰**: 회사 이벤트 / 규제 정책 / 산업 경쟁사 / 시장 심리 (셀사이드+인플루언서+남향 자금)
- **나열보다 귀인 우선**: 모든 뉴스를 늘어놓는 게 아니라 "어느 이벤트가 이번 주가 급변에 값하는가"를 판단
- **성격 판단 강제**: 가치 이벤트 / 심리 변동 / **진짜 원인 불명** / 혼합 — 그중 "진짜 원인 불명"이 가장 가치 있는 출력 (내부자 선행 매매 가능성)
- **명확한 행동 권고**: 심층 연구 발동 여부, 논거 재검토 필요 여부, 단순 관찰 여부 등

**다른 Skill과의 차이**:
| 상황 | 무엇을 쓰나 |
|------|------|
| 완전 투자 리서치 (시간 단위) | `/investment-team` 또는 `/investment-research` |
| 재무제표 정독 | `/earnings-review` |
| 장기 논거 추적 | `/thesis-tracker` |
| **주가 급변 10분 귀인** | **`/news-pulse`** |

**출력 예시 발췌** (텐센트 4/17-5/01 실측, 14일 -10.47%):

> #### 한 줄 귀인
> 이번 -10.47% 하락은 약 70-80%가 자금면+심리면 주도(자사주 매입 침묵기 + 남향 매도 + 섹터 베타 + AI 내러티브 탈취)이고, 20-30%가 AI 투자 2배 증가의 이연 소화가 담당한다 — **펀더멘털 악재 없음**, 셀사이드는 매수 컨센서스 유지, 성격상 "유동성+심리형 조정"이지 가치 이벤트가 아님.
>
> #### 급변 귀인표
>
> | 후보 설명 | 추정 기여 | 신뢰도 |
> |---------|--------|--------|
> | 자사주 매입 침묵기 소멸(구조적, 5/13 실적 전) | -3% ~ -4% | 높음 |
> | 남향 자금 텐센트 순매도 전환 | -2% ~ -3% | 높음 |
> | AI 내러티브 경쟁사에 탈취(DeepSeek V4/Qwen3.6/문샷 1T) | -1% ~ -2% | 중 |
> | 섹터/매크로 베타(유가+지정학+Fed Warsh 매파) | -2% ~ -3% | 높음 |
> | 1분기 실적 전 리스크 회피 | -1% ~ -2% | 중 |
> | 펀더멘털 악화 | **0%** | 매우 높음 (배제) |
>
> #### 성격 판단: ✅ 혼합형
> 70% 자금면/심리면 + 20% AI 장기 내러티브 우려 + 10% 1분기 실적 전 불확실성
>
> **핵심 반증**: 돤융핑 4/8 텐센트 풋 매도(강세); 셀사이드 24개사 컨센서스 Strong Buy; 넷이즈 4/30 역행 +2%(게임 산업 문제 배제); 텐센트가 항셍테크 대비 7%p 언더퍼폼(항셍테크는 월간 오히려 +4%).

호출 방식:

```
/news-pulse 카카오
/news-pulse 삼성전자 12% 하락 일주일 내
/news-pulse 미호요
```

## 설계 철학

### 4대 대가 방법론 융합

```
              ┌──────────────────┐
              │    돤융핑         │
              │  "옳은 사업"      │
              │  비즈니스 모델 본질│
              └────────┬─────────┘
                       │
    ┌──────────────────┼──────────────────┐
    │                  │                  │
    ▼                  ▼                  ▼
┌────────┐     ┌──────────┐      ┌────────┐
│  버핏   │     │   멍거    │      │  리루   │
│ 해자    │     │ 역발상    │      │문명트렌드│
│ 안전마진│     │ 리스크목록│      │패러다임전환│
│ 경영진  │     │ 편향자가점검│    │ 산업가치│
└────────┘     └──────────┘      └────────┘
```

4대 대가는 단순한 분업이 아니라 **서로 도전하도록** 설계되었습니다.
- 돤융핑이 "좋은 사업"이라 하면, 멍거가 "어떻게 죽는가"를 묻는다
- 버핏이 "충분히 싸다"고 하면, 리루가 "10년 후에도 있는가"를 묻는다
- 당신이 얻는 것은 네 개 보고서의 짜깁기가 아니라, 네 가지 사고방식의 충돌이다

### 금융 엄밀성 도구 (`tools/financial_rigor.py`)

| 기능 | 명령 | 해결하는 문제 |
|------|------|-----------|
| **시가총액 검산** | `verify-market-cap` | 주가×총주식수 정밀 계산, 단위 오류 검출 |
| **밸류에이션 검산** | `verify-valuation` | PE/PB/ROE/FCF Yield 정밀 십진수 계산 |
| **다출처 교차 검증** | `cross-validate` | N개 출처의 동일 데이터 자동 대조, 허용오차 초과 시 경보 |
| **3개 시나리오 밸류에이션** | `three-scenario` | 낙관/중립/비관 목표가 정밀 계산 |
| **벤포드 법칙 검출** | `benford` | 재무 데이터 선두 숫자 분포 이상 검출 |
| **정밀 계산기** | `calc` | 임의의 재무 수식 정밀 계산, LLM 암산 대체 |

`--currency` 옵션으로 **한국 원화(KRW)·미국 달러(USD)·홍콩달러(HKD)·위안화(CNY)** 등 통화를 지정할 수 있어, 한국·미국 종목 검산도 그대로 지원합니다.

**설계 원칙**: 모든 계산은 Python `decimal.Decimal`(정밀 십진수)을 사용하며 `float`(부동소수점 근사)을 쓰지 않습니다. `0.1 + 0.2 = 0.3`은 금융 시나리오에서 실패하면 안 됩니다.
